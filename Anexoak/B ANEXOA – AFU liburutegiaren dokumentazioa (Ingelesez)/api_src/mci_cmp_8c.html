Control the component codes with the asset codes and set a warning if any used.

Controls that the used component codes are different of the assigned asset codes for the treated instrument. It sets a warning message for each component code equal to any asset assigned to the instrument.

Parameters
LdCmpList of component codes. At least, it should have a "CmpCod" column.
LdAssList of assigned asset to the instrument. At least, it should have a "MciAst" column

int MCIcmpCtl ( void *  LdCmp,
void *  LdAss,
int *  Ctn 
)

Payoff schedule component control.

Payoff schedule component control. Done controls:

  • The only component that can be duplicated is the empty string "" because in all subcomponents in the matrix have "" as component
  • If any amount type is 'tim', control the correctness of the selected time dependent table when it is a standard one.
  • The component code cannot be same code as any existent asset
Parameters
LdCmpComponents list
LdAssAssets list
Ctn(out) Boolean. Continue with the treatment (1) or not (0)
Returns
Error code (0: OK)
