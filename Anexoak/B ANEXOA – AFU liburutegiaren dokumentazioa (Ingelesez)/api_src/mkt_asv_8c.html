Detailed Description

Market data calculations for ASV curves.

Market data calculations for AsSet Volatility curves. See mkt.c for details

  • - Must remeber that MKTasvCal function needs the yield curves to make calculations. This means that MKTyldCal must be done to call this function. In order to avoid managging not needed curves MKTasvChk should be called before MKTyldChk
  • Object: Asv, list of asset volatility.
  • Name Type Origin

Calculate the date columns for one curve.

Controls ASV curves.

Controls ASV curves. We check that all the required curves have been loaded with correct values. All curves that are not used will be destroyed, so that we don't need to calculate it.

  • In the stock object (LdMci) all asv curves come in the "Asv" column.
