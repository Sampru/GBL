Detailed Description

Interface with engines curve and curve points definition windows.

Header file
emkt_def.h
Acronyme
ECRV
Library
emkt

Interface with engines curve and curve points definition windows. Functions to deal with window to define a curve (CrvDev, sdt) and to define the points of the curve (CrvPnt, MIE).

Author
Edurne GORDOA - 09/Apr/2002

Specific control of curve definition window.

Specific control of curve definition window

  • If it is a creation or modification, check that there are points defined for the curve
  • If it is a deletion, make sure that the curve is not being used
  • For CDS and CPS curves, make sure that there is not already another curve with same entity and prelation order

  • For CPS curves, check the way of generate CPS curve is defined
