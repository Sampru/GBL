Detailed Description

MarKeT data for INdeX.

Header file
mkt_def.h acronym : MKinx
Library
mkt

MarKeT data for INdeX

  • Object: Inx, list of index prices. Sorted by cod
  • Name Type Origin Description
  • cod CTR_TYPE mkt_mcd.cod Index code
  • dev DEV_TYPE bsk.dev Currency
  • Pay DEV_TYPE bsk.Pay Country of the index
  • Mai char [4] bsk.Mai Main index of the country
  • imc MCD_TYPE MKTmcdSetImc Internal market code
  • val DBL_TYPE mkt_dta.val, Typ = 'Inx' Price
  • fuv DBL_TYPE MKTinxCal FIT Unit Value
  • sta ITR_TYPE mkt_dta.sta Price status

  • src ITR_TYPE mkt_dat.src Price source

Load index prices into a single data structure, LdMkt

  • It can take market data from current or historic dates.
Parameters
CtxDatabase connection
DatPrices date. If NULL, take current market data.
ProProfile
TrtOnly for historic data, treatment for which the historic market data must be recovered in case of FO book. If it is empty or NULL, revaluation treatment will be used.
MktHisOnly for historic data, flag to get all market data (mkt_his_all) or only used prices (mkt_his_use)
LdMkt(in/out) Market data structure (Add object "Inx")
Returns
Error code (0: OK)

Load historic index prices into a single data structure, LdMkt

  • If the database connection is MO, it will take historic market data
  • If the database connection is FO, it will take historic market data for the given treatment, for example, revaluation.
