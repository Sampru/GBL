int AFUccyGetDta ( void **  LdAfuCcy)

Recover the value of static list 'lcLdCcyDes', that is, information of afuCcy table.

  • Recover the value of static list 'lcLdCcyDes', that is, information of afuCcy table.
  • The function returns a list with:
  • - dev
  • - den
  • - EurDev
  • - pip
  • - qus
  • - cff
  • - quf
  • - qua
  • - LibQntOri: the containing of 'LibQnt' field of afuCcy table
  • - LibCtgOld: the containing of 'LibCtg' field of afuCcy table
  • - PrcDev
  • - EchCtg : the containing of 'RndQnt' field of afuCcy table
  • - EchQnt : the numeric value of 'LibQnt' field of afuCcy table (ths->3, mil->6, unt->0)
  • - RndQnt : the same value as EchQnt + RndQnt field of afuCcy table
  • - LibQnt : label for the currency, k<dev>, M<dev>,...that corresponds to LibQnt field
  • - LibCtg : label for the currency, k<dev>, M<dev>,...that corresponds to LibCtg field
Parameters
LdAfuCcy(out) List containing the same data as 'lcLdCcyDes' static list. It contains information of afuCcy table
Returns
Error code (0: OK; -1: Error)
Author
Josune BIKUNA - 29/Jan/2015
