Detailed Description

Market data - Correlation matrix between entities.

Market data - Correlation matrix between entities

  • For the time being, correlation matrix is not a real market data, there are not historic values neither different values per profiles, but we will treat it as if it were.
  • Must remember that MKTcorChk uses CDS market data and MKTcdsChk uses CDS baskets. This means that correct order in checking process is MKTcdsBskChk, MKTcdsChk, MKTcorChk.
  • See mkt.c for details.
  • Datastru:
  • Object: Cor, list of baskets with correlations between entities and their simul mtx.
  • Name Type Origin Description
  • Ent1 CIE_TYPE RefCorr.Ref1 Entity 1
  • Ent2 CIE_TYPE RefCorr.Ref2 Entity 2

  • Cor DBL_TYPE RefCorr.Cor Value

Build the correlation matrix using the basket correlation value.

Build the correlation matrix. The correlation value between

  • the same entities is 1.0. We dont use the curve correlation of basket.
  • To build the matrix we use the same correlation value between entities.
  • If we want to use specifics correlations values between entities the
  • input correlation value must be 0.0 >= Cor > 1.0, and we search the
  • value from MKTcorFndCor function.
Parameters
LdMktList of market data

  • Num: Deal number.

  • Dev: Currency.
