Detailed Description

Market data - Equity prices.

Market data - Equity prices (See mkt.c for details.)

  • Datastru:
  • Object: Epr, list of Equity current PRices. Sorted by cod
  • Name Type Origin Description
  • cod CTR_TYPE mkt_mcd.cod Equity code (contract)
  • dev DEV_TYPE MKTeprChk () Equity currency
  • val DBL_TYPE mkt_dta.val Equity price
  • fuv DBL_TYPE MKTeprCal () Fox Unity Value
  • mrk ITR_TYPE MKTeprChk () Mark
  • Object: Epm, list of Equity Prices of Multiple dates. Sorted by cod and date
  • Name Type Origin Description
  • dat CDA_TYPE mkt_dta_his.dat Date
  • cod CTR_TYPE mkt_mcd.cod Equity code (contract)
  • dev DEV_TYPE MKTeprGetMul () Equity currency
  • val DBL_TYPE mkt_dta_his.val Equity price
  • fuv DBL_TYPE MKTeprCal () Fox Unity Value
  • mrk ITR_TYPE MKTeprChk () Mark
  • Object: Epd, list of Equity Prices Dates. Sorted by date
  • Name Type Origin Description
  • dat CDA_TYPE mkt_dta_his.dat Date

int MKTeprCalYld ( void *  LdMkt)

Calculate the dividend_yield of each equity contract.

Calculate the dividend_yield of each equity contract We need to calculate the dividend_yield of each contract. To do it, we have to:

  • For each contract:
    • Get contract's events
    • Get 'Cash dividends' (evncdi)
    • Sort (Descending) contract's events taking into account the value date (dt3)
    • Take the list of dates of the last known event date's year. For example: We have events for these dates:
      • 03/02/2004
      • 16/06/2004
      • 03/02/2005
      • 16/06/2005 The last event date that we know is 16/06/2005. And then the list = (03/02/2005, 16/06/2005). Note: The last date that we know it is possible to be future even past
    • Sum Amount per share (fl1) column's value —> A
    • Take the market value of the contract —> B
    • And dividend_yield = A/B
    • We need in percentage, so (A/B)*100
Parameters
LdMktList of market data
Returns
Error code (0: Ok)
