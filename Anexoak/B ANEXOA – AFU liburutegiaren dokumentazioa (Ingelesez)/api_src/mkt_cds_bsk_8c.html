Detailed Description

CDS basket or basket references treatments.

Header file
mkt_def.h
Acronyme
MKTcdsBsk
Library
mkt

CDS basket or basket references treatments. The basket is created with a number of

  • reference. Each reference have a CDS curve.
  • The basket is also related with a CDSI curve.
  • The basket is also related with a correlation value or correlation curve.
  • And we can define a revaluation method of basket.
Author
Manuel SANMARTIN - 07/Dec/2005
  • - Data Structure.
  • - Object: CdsBsk, list of CDS baskets
  • Name Type Origin Description
  • Bsk CUR_TYPE mktCdsBsk.Cod Basket name
  • Idx BOO_TYPE mktCdsBsk.Idx Flag of index or not
  • Rrt DBL_TYPE mktCdsBsk.Rrt Basket recovery rate
  • Frt DBL_TYPE mktCdsBsk.Frt Fixed recovery rate
  • BskCor DBL_TYPE mktCdsBsk.BskCor Basket correlation
  • Bco DBL_TYPE mktCdsBsk.Bco Basket correlation curve
  • Mth CUR_TYPE mktCdsBsk.Mth Revaluation method
  • Crv CUR_TYPE mktCdsBsk.Crv CDSI curve related with the basket
  • LdRef DYN_VOID MKTcdsBskGetBsk () List of references of the basket excluded
  • default values
  • Mtx MATRIX MKTcdsBskCal () Matrix of the references of baskets
  • mrk ITR_TYPE MKTcdsBskChk () Mark column
  • Alv DBL_TYPE MKTcdsBskCal () Basket weight (sum of reference weights)
  • LdRef: List of references of basket
  • Name Type Origin Description
  • Frr DBL_TYPE mktCdsBskRef.Frr Entity recovery rate
