char *  StlDes,
char *  StlCod,

)

Get the results format type according to a given results measure.

Get the results format type according to a given results measure. For getting the results format two different parameters can be used, in the way that at least one one them must be informed:

  • - the description of the results measure
  • - the code of the results measure
  • By the moment, the possible results format types are:
  • - MCI_RFM_BY_ASS: results by asset
  • - MCI_RFM_BY_MCI: results by MC intrument
Parameters
CtxDatabase context
StlDesDescription for the result measure for which the results format is asked. Optional if the code is given
StlCodCode for the result measure for which the results format is asked. Optional if the description is given.
RfmResults format type. It could have the values: MCI_RFM_BY_ASS (by asset), MCI_RFM_BY_MCI (by MC instrument). This string must be previously reserved before calling to this function
Returns
Error code (0 if OK).
