Get the curve type code from a curve entry screen identifier.

Get the curve type code from a curve entry screen identifier. The treated codes are:

  • "adycrv" || "mkhadycrv" –> "Ady"
  • "afuBco" || "mkhBco" –> "Dyc"
  • "afuCdiVol" || "mkhCdiVol" –> "Siv"
  • "afuCdsVol" || "mkhCdsVol" –> "Dsv"
  • "afuDvyCrv" || "mkhDvyCrv" –> "Bco"
  • "afuEprCrv" || "mkhEprCrv" –> "Epc"
  • "afuEprVol" || "mkhEprVol" –> "Epv"
  • "afuFrxVol" || "mkhFrxVol" –> "Fxv"
  • "afuInxCrv" || "mkhInxCrv" –> "Ixc"
  • "afuInxVol" || "mkhInxVol" –> "Ixv"
  • "asvcrv" || "mkhasvcrv" –> "Asv"
  • "cdscrvprx" || "mkhcdscpr" –> "Cds"
  • "cdsicrvprx" || "mkhcdsicpr" –> "Cdsi"
  • "fspcrv" || "mkhfspcrv" –> "Fsp"
  • "ylvcrvprx" || "mkhylvcpr" –> "Ylv"
  • .... the rest –> "Yld"
