Header file

Acronyme
MKTbco (MarKeT Data Base COrrelation curves).

Market data - Base correlation curves. See mkt.c for details.

  • - Data Structure.
  • - Object: Bco, list of base correlation curves
  • Name Type Origin Description
  • cod CUR_TYPE crv_des.crv Curve name
  • dev DEV_TYPE crv_des.dev Currency
  • LdCrv DYN_VOID MKTbcoGetBco () List of points

  • dat CDA_TYPE MKTbcoCal () Calculation date
  • LdCrv: Curve, list of points
  • Name Type Origin Description

Control base correlation curves.

Control base correlation curves. Check that all the required curves have been loaded with correct values. All curves that are not used will be destroyed, so that we don't need to calculate it.

  • In the stock object (LdTra) all curves come in the "bco" column.

Load historic base correlation curves into a single data structure.

Load historic base correlation curves into a single data structure. We may have some indications in LdMkt on which curves are necessary.

  • If the database connection is MO, it will take historic market data
  • If the database connection is FO, it will take historic market data for the given treatment, for example, revaluation
Parameters
CtxDatabase context
DatCalculation date
ProProfile
TrtTreatment for which the historic market data must be recovered in case of FO book. if it is empty or NULL, revaluation treatment will be used.
MktHisFlag to get all market data (mkt_his_all) or only used prices (mkt_his_use)
LdMkt(in/out) Market data structure
Returns
Error code (0: OK)
