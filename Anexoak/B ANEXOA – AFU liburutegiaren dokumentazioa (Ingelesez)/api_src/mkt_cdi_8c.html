Calculate the date columns for one curve.

Calculate the date columns for one curve.

  • bdt: Begin date
  • edt: End date
  • pdt: Payment date
  • nbd: days from bdt till edt.
  • nbp: days from bdt till pdt. Get the IMM date next to CrvBdt as reference date. If CalDat is bigger than the curve begin date date, get next open date to CalDat as begin date for all points; otherwise, get CrvBdt as begin date for them. From that, move the maturity to get end date of each point.
Parameters
CalDatCalculation, revaluation date.
CrvBdtCurve begin date
DevCurve currency
LdCrv(in/out): LdCrv (plg, mat, unt) -> LdCrv (nbd, bdt, nbp, edt, pdt)
Returns
Error code (0: OK)
Author
Basilio TAMAYO - 03/Nov/2006. Recovered from previous versions.

Calculate tweaking of CDSI curves.

Calculate tweaking of CDSI curves.

  • For each CDSI curve
  • + For each point
  • - Duplicate the complete CDSI curve
  • - Shift a given value the value of the point of the CDSI curve
  • - Calculate the CPI curve associated with the shifted CDSI curve

  • - Save the CPI and CDSI shifted curves

    Parameters
    CalDatCalculation date
    LdMkt(in/out) Initial market data. Must have (checked and calculated) at least CDS curves, CPS curves, yield curves and recovery rates. Will add "CpiTwe"
