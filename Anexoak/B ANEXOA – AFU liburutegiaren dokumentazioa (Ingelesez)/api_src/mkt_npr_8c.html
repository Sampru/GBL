Detailed Description

Market data - Fund prices.

Market data - Fund prices (See mkt.c for details.)

  • Datastru:
  • Object: Npr, list of fuNd current PRices. Sorted by cod
  • Name Type Origin Description
  • cod CTR_TYPE mkt_mcd.cod Fund code (contract)
  • dev DEV_TYPE MKTnprChk () Fund currency
  • val DBL_TYPE mkt_dta.val Fund price
  • fuv DBL_TYPE MKTnprCal () Fox Unity Value
  • mrk ITR_TYPE MKTnprChk () Mark
  • Object: Npm, list of fuNd Prices of Multiple dates. Sorted by cod and date
  • Name Type Origin Description
  • dat CDA_TYPE mkt_dta_his.dat Date
  • cod CTR_TYPE mkt_mcd.cod Fund code (contract)
  • dev DEV_TYPE MKTeprGetMul () Fund currency
  • val DBL_TYPE mkt_dta_his.val Fund price
  • fuv DBL_TYPE MKTnprCal () Fox Unity Value
  • mrk ITR_TYPE MKTnprChk () Mark
  • Object: Npd, list of fuNd Prices Dates. Sorted by date
  • Name Type Origin Description
  • dat CDA_TYPE mkt_dta_his.dat Date
