Detailed Description

Market data - Commodity prices.

Market data - Commodity prices (See mkt.c for details.)

  • Datastru:
  • Object: Com, list of COModity current prices. Sorted by cod
  • Name Type Origin Description
  • cod CTR_TYPE mkt_mcd.cod Commodity code (contract)
  • dev DEV_TYPE MKTcomChk () Commodity currency
  • val DBL_TYPE mkt_dta.val Commodity price
  • fuv DBL_TYPE MKTcomCal () Fox Unity Value
  • mrk ITR_TYPE MKTcomChk () Mark
