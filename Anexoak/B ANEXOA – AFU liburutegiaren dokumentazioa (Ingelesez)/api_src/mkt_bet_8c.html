Detailed Description

MarKeT data for BETa.

MarKeT data for BETa

  • Object: Bet, list of beta prices. Sorted by cod (ctr)
  • Name Type Origin Description
  • cod CTR_TYPE mkt_mcd.cod First asset
  • pnt CTR_TYPE mkt_mcd.pnt Second asset
  • dev DEV_TYPE mkt_mcd.dev Currency
  • val DBL_TYPE mkt_dta.val, Typ = 'Bet' Price
  • itr ITR_TYPE MKTbetCal Number of iterations to calculate the beta

  • mrk ITR_TYPE MKTbetCal Mark

int MKTbetFndAssBet ( void *  LdBet,
char *  Ass,
void *  LdVer,

double *  Bet 
)

For a given asset, search the most suitable vertex and its correspondent beta.

For a given asset, search the most suitable vertex from a list of vertexes and its correspondent beta

  • If the asset itself is in the list of valid vertexes, the beta is 1.0 and the vertex is the asset itself.
  • If not, search in the list of betas those with the asset against any vertex in the lists.

  • If there are more than one vertex in the list of betas, get the one with less iterations. NOTE : IS OBLAGATORY that the beta list (LdBet) must be have the all betas defined (With price or not price). If this list contain only the beta with its prices, this function visualized always the message: "Missing beta definition" For obtain the all beta list with its prices or not prices call to MKTbetGetAllBet function

    Parameters
    LdBetList of calculated betas
    AssFirst asset of the searched beta
    LdVerList of valid vertexes of the searched beta. Vertex column (cod) must be the first column
    Ver(out) Selected second asset. It reserves memory to save it; so, it must be liberated.
    Bet(out) Beta price between both assets
    Returns
    Error code (0: OK)
