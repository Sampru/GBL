Calculate CPSI curve from one CDSI curve using bisection method.

Calculate CPSI curve from one CDSI curve using bisection method.

  • CPSI: Index Solvency Probability Curve.
  • We need a CDSI curve from one issuer (reference), the value date and the yield curve.
  • For each CDSI curve point we need to know the maturity date, the basis, the spread and payment period.
  • This function calculates the lambdas (CPSI curve) values for one CDSI curve.
  • The CPSI curve will have the next structure : bdt - edt - mat - unt - lmb - pso – bdt: Init date of the curve – edt: Maturity date (Infinity in the last interval) – mat: Maturity value of the point of the CDSI curve used to calculate. – unt: Unit of the maturity of the point of the CDSI curve used to calculate. – lmb: Lambda – pso: Solvency probability, not cumulative
