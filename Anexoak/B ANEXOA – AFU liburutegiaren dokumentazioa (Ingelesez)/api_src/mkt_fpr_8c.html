Detailed Description

Market data - Future prices (future, MMfuture, Ifuture, ...)

Market data - Future prices (future, MMfuture, Ifuture, ...).

  • See mkt.c for details.
  • Datastru:
  • Object: Fpr, list of future prices. Sorted by ctr
  • Name Type Origin Description
  • cod CTR_TYPE mkt_mcd.cod Future contract
  • val DBL_TYPE mkt_dta.val Future price
  • vad DBL_TYPE CVTprice(mkt_dta.val) Future price, in decimal

  • dlt DBL_TYPE CVTprice(mkt_dta.val) Delta
