Market data - Recovery rate.

Header file
mkt_def.h
Acronyme
MKTrrt (MarKeT Data Recovery RaTe)
Library
mkt

Market data - Recovery rate

  • For the time being, recovery rate is not a real market data, there are not historic values neither different values per profiles, but we will treat it as if it were.
  • Must remeber that MKTrrtChk uses CDS market data and MKTcdsChk uses CDS baskets. This means that correct order in checking process is MKTcdsBskChk, MKTcdsChk, MKTrrtChk.
  • See mkt.c for details.
  • Datastru:
  • Object: Rrt, list of recovery rates. Sorted by cod
  • Name Type Origin Description
  • cod CUR_TYPE EntRecRat.Cod Code
  • ent CIE_TYPE EntRecRat.Ent Entity
  • dev DEV_TYPE EntRecRat.Dev Currency
  • pre CUR_TYPE EntRecRat.PreOrd Prelation order

  • val DBL_TYPE EntRecRat.RecRat Value
