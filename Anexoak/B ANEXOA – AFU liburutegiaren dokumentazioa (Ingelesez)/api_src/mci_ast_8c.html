Detailed Description

functions for asset definition for a MC instrument

In a MC simulated instrument there are assets needed for execution. This file contains the function that handles these assets.

Returns a LL list with all variable parameters grouped in a string.

It returns a LL list with one line for each instrument-deal. The variable parameters are grouped in a string and saved in a new column: StrPvb. Get the list of assets which are used into the one instrument.

Get the list of assets which are used into the one instrument. It returns a list with all data of the afuAst table adding the code of the instrument (MciCod). It also returns the curve codes saved in different columns depending on the market data type. For equity assets, it depends also in the diffusion model.

  • equity (lp,np) –> column for fwd. curve: Epc; column for volat.curve: Epv
  • equity (ls,lv) –> column for fwd. curve: Dyc; column for volat.curve: Epv
  • index –> column for fwd. curve: Ixc; column for volat.curve: Ixv
  • spot –> column for fwd. curve: Fsp; column for volat.curve: Fxv
  • interest rate –> column for fwd. curve: crv; column for volat.curve: Ylv
  • commodities –> column for fwd. curve: coc; column for volat.curve: cpv
Parameters
CtxDatabase connexion
LdAlsDiffusion books list
LdPvbList of the variable parameters (for the case of the DLE)
MciCodCode of the Monte Carlo instrument
IntCodInternal code of the Monte Carlo instrument
TcNumDeal number to get its specifics assets
LdAss(OUT) Resultant Asset list. "SrcAss" has the generic asset code, "AssCod", its correspondent concrete name
Returns
Error code (0: OK)

Verify if a market datum is used as a MC instrument's asset.

Verify if a market datum is used as a MC instrument's asset. The datum used to find a market datum depends on the field passed by argument 'Fld':

  • cod: market datum code.
  • imc: internal market code (IMC). If the market datum is used in an asset, it sets a warning and returns a 1 in the 'Use' argument. Otherwise, it returns a zero. If there are multiples assets that use the market datum, it only shows the code of the first one.
Parameters
CtxDatabase context
LdDtaData list
MktTypMarket type used to set the IMC. Take from the KeyByTyp structure
FldField of mkt_mcd used to select a market datum (cod, imc...)
Use(out). (1) if the market datum is used in an asset and (0) if not.
Returns
Error code (0 if OK).
