Return a list with different pair of correlation between a list of assets.

Return a list with different pair of correlation between a list of assets.

Parameters
LdAssAsset List. It must be sorted by asset code and It must has "Ass", "NumKop", "MciCod", "Dev" columns defined where:
  • Ass: Asset code.
  • NumKop: Deal number. It can be '' but not NULL.
  • MciCod: Code of instrumen. It can be '' but not NULL.
  • Dev: Currency.
LdCor(out) Correlation prices list. It must exist and it has to have the follogin columns defined, "AssOne", "AssTwo", "Num", "MciCod", "Dev", "AssOneUmc", "AssTwoUmc" where :
  • AssOne: Asset one code.
  • AssTwo: Asset two code.
  • Num: Deal number.
  • MciCod: Code of instrument.
  • Dev: Currency.
  • AssOneUmc: Umc of AssOne.
  • AssTwoUmc: Umc of AssTwo.
Returns
Error code (0: OK)

Calculate the volatility with the historic prices.

Calculate the volatility with the historic prices

Parameters
CtxDatabase context
DatCalculation date
ProProfile for getting the market data
FlgTdyFlag to indicate is today or historic calculation
VolCalFrqFrequency for calculating the volatilies. Possible values:
  • mvcVolCalFrqDly: (daily) the volatilities are calculated for each calculation date using the prices that correspond to that date
  • mvcVolCalFrqWek: (weekly) the volatilities are calculated just once a week, when the calculation date is FRIDAY, and for the rest of days the calcualtion is not done and the volatilities are the same as the previous date.
Zcp4YldFlag that indicates if for yield curves, the zero coupon values must be used (1) or directly the values of the points (0)
SpoRefDevReferency currency for getting the spots. All the spots will be given against this currency. This paramter is optional, and by defalt the "USD" currecy is used.
EqeImpDirImportation directory for the equity events files that impact in the Vol&Cor matrixes calculation, for instance the equity events splits done over the same contract. In case this parameter is not valid (NULL or empty), the management of these files will be ignored.
LdAss(in/out) Assets with their market data prices
LdVol(out) Assets with their historic volatilities
Returns
Error code (0: OK)
