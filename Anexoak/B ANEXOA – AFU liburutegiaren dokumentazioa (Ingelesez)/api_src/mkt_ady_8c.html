Detailed Description

Market data calculations for ADY curves.

Acronyme
MKTady (MarKeT data Asset Divident Yield)

Market data calculations for Asset Divident Yield curves. See mkt.c for details

  • - Must remeber that MKTadyCal function needs the yield curves to make calculations. This means that MKTyldCal must be done to call this function. In order to avoid managging not needed curves MKTadyChk should be called before MKTyldChk
  • Object: Ady, list of asset divident yield.
  • Name Type Origin
