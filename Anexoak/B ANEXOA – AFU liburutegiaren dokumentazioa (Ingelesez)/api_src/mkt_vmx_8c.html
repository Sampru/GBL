Detailed Description

Market data: Volatility MatriX.

Management of the volatility matrixes (definition and data insertion).

  • Object: Vmx, list of volatility matrixes. Sorted by Cod
  • Name Type Origin Description
  • Cod CTR_TYPE mkt_mcd.cod Volatility matrix
  • Ccy DEV_TYPE afuVolMtx.Ccy Currency
  • Prd PRD_TYPE afuVolMtx.Prd Product
  • Xax char (3) afuVolMtx.Xax Concept saved into the X axis
  • Yax char (3) afuVolMtx.Yax Concept saved into the Y axis
  • Zax char (3) afuVolMtx.Zax Concept saved into the Z axis
  • LdXax DYN_VOID MKTvmxCal () List of X axis points
  • LdYax DYN_VOID MKTvmxCal () List of Y axis points
  • LdCub DYN_VOID MKTvmxGet () List to the cube
  • dat CDA_TYPE MKTvmxGetVmx () Calculation date
  • List: LdXax, X axis points. It can contain strike or maturity points.
  • Name Type Origin Description
  • LdXax, when X axis contains the strike
  • Mtx CTR_TYPE mkt_mcd.cod Volatility matrix
  • Stk DBL_TYPE MKTvmxCal () Strike
  • LdXax, when X axis contains the init dates
  • Mtx CTR_TYPE mkt_mcd.cod Volatility matrix
  • Mat INT_TYPE MKTvmxCal () Date value, e.g. 90
  • Unt INT_TYPE MKTvmxCal () Term unit, e.g. "D"
  • Edt CDA_TYPE MKTvmxCalMatAxs () Date for the maturity
  • Nbd INT_TYPE MKTvmxCalMatAxs () Number of days till Edt
  • List: LdYax, Y axis points. Usually (maybe always) it will be the init dates
  • Name Type Origin Description
  • Mtx CTR_TYPE mkt_mcd.cod Volatility matrix
  • Mat INT_TYPE MKTvmxCal () Date value, e.g. 90
  • Unt INT_TYPE MKTvmxCal () Term unit, e.g. "D"
  • Edt CDA_TYPE MKTvmxCalMatAxs () Date for the maturity
  • Nbd INT_TYPE MKTvmxCalMatAxs () Number of days till Edt
  • List: LdCub, cube, list of Z axes and its matrixes
  • Name Type Origin Description
  • Mtx CTR_TYPE mkt_mcd.cod Volatility matrix
  • Mat INT_TYPE MKTvmxCal () Date value, e.g. 90
  • Unt INT_TYPE MKTvmxCal () Term unit, e.g. "D"
  • LdPts DYN_VOID MKTvmxCal () List of values list
  • LdMtx DYN_VOID MKTvmxCal () List of values matrix
  • List: LdPts, values list
  • Name Type Origin Description
  • cod CUR_TYPE MKTvmxCal () Matrix code
  • imc RIC_TYPE MKTvmxCal () Internal point imc code
  • val DBL_TYPE MKTvmxCal () Point value
  • sta ITR_TYPE MKTvmxCal () Status
  • src DBL_TYPE MKTvmxCal () Source
  • LdMtx: Matrix, list of Z axes and its matrixes
  • Name Type Origin Description
  • b0 DBL_TYPE MKTmkMtx () 1st data column
  • b1 DBL_TYPE MKTmkMtx () 2nd data column

  • b2 ...
