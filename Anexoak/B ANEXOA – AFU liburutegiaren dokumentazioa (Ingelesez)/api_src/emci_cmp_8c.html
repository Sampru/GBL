int EMCIcmpCtl ( void *  LdObj,
int *  Ctn,
int *  Don 
)

Payoff schedule component control.

Payoff schedule component control. Done controls:

  • The only component that can be duplicated is the empty string "" because in all subcomponents in the matrix have "" as component
  • If any amount type is 'tim', control the correctness of the selected time dependent table when it is a standard one.
  • The component code cannot be same code as any existent asset
Parameters
LdObjObjects list.
Ctn(out) Boolean. Continue with the treatment (1) or not (0)
Don(out) Boolean. All specific verification done (1) or generic needed (0)
Returns
Error code (0: OK)
