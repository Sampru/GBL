Calculate the selected instruments.

Calculate the selected instruments passed in the "LdMci" list. Notes:

  • Only points with valid data of the market data curves are passed to FEA calculation function.
  • The results are saved in the same book where the deal is saved.
  • When "lmm" model is defined, the 'nfactors' criterion is let empty to indicate FEA to use the default value.
Parameters
CtxDatabase connection
DvaValue date of the simulation.
LdMktMarket data necessary list necessary to MonteCarlo simulation.
LdMciList necessary structure to MonteCarlo simulation.
LdCalList with the elements that will be calculated
SavFlag to save or not in database the results of the calculation.
LdMciRes(out). Resultant list of MonteCarlo simulation.
Returns
Error code (0: OK)
