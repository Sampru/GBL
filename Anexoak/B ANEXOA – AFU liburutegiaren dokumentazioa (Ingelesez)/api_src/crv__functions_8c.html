Detailed Description

Curve functions.

Acronyme
CRV
Library
mkt

Curve functions. Functions to deal with window to define a curve (CrvDev) and to define the points of the curve (CrvPnt).

  • Add a list of predefined points.
  • These points can be of type Overnight, Tom. Next, End of Year, cash, IRS or OIS.
  • And every point have the following attributes:
  • - pty: Point type, one of the above mentioned
  • - cod: Point code, identifies each point
  • - leg: Number of leg (usually 0)
  • - met: Interest calculation method: proportional or actuarial
  • - bas: Basis of date calculation (ACT/ACT, ...)
  • - fqm: Coupon frequency maturity (0 means no coupons)
  • - fqu: Coupon frequency unit
  • - brk: Break coupon (Short: make two coupons; Long: join in one)
  • - pos: Coupon position, begin or end of period
  • - adj: If there is date adjustment for closed dates
  • - pfl: Adjustment method, following or preceding open day
  • - plg: Payment lag, number of days
  • - quo: Quotation, rate or price
  • - per: Future period, number of days

  • - ffx: Initial date mode for future points (FIXED31, FIXED33, LIBOR-2, LIBOR-1)

    Author
    Basilio TAMAYO - 18/Feb/2000
