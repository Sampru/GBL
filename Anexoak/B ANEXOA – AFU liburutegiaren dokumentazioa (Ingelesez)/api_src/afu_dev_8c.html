Convert amounts in marked lines from currency units.

Given a list with a currency column and one or more amount columns, divide every amount by the currency coefficient. This function converts amounts from currency units to financial units (e.g.: from EUR to kEUR) Only if the lines are marked (value in ColMrk column != 0

Convert amounts in a list.

Given a list with a currency column and one or more amount columns, divide or multiply every amount by the currency coefficient.

  • Depending on the convertion factor we are going to pass the amount from financial units to currency units (e.g.: from kEUR to EUR) or viceversa (e.g.: from EUR to kEUR)

  • It will only be converted lines that are marked.

    Parameters
    CtxDatabase connection
    LdList to be treated
    CnvFactconvertion factor. +1.0 to pass from financial units to currency units -1.0 to pass from currency units to financial units
    ColDevCurrency column identifier
    ColMrkMark column to identify the lines whose values must be converted. -1 for all.
    NbColNumber of column to be multiplied
    ColArrArray with the amount column indexes
    Returns
    Error code (0: OK)
