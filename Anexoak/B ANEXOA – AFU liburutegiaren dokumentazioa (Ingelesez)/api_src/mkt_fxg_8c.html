int MKTfxgCalAcc ( void *  Ctx,
void *  LdCC,
char *  Fxg,
char *  ValDat 
)

Calculation of the accrued coupon from a list with TMP.

Given a list where in every line there is a fixing data, begin date, end date and a rate, this function calulates the accrued from the last date (the newest) to the others. The calculated value isn't the CC (aCcruec Cupon) but CC * NbDays / (Bas / 100). This value will be stocked in a column called 'CC', if it doesn't exist it will be created. Also get in a column the equivalent daily 'CC' ('Column EquCC').

  • Name of the columns: bdt, edt, rat
  • Note: if dev and fdv are not identical, it will be a problem with ValDat if dev is opened but fdv is closed, because we'll never get a period that finish in ValDat
Parameters
CtxDatabase context
LdCC(in/out) List with input information and for stock calculated values
FxgFixing code
ValDatValue date (RemDec). Date for the end date in the last date in the list
Returns
Error code (0: OK)

Copy the fixing rates of a fixing day into other fixing days.

Copy the fixing rates of a fixing day into other fixing days If between the original date DatOrg and DatDst are more than one day, the fixings rates will be copied for all those days. If CrvSet and Dev are empty all the fixings rates will be copied, but the fixing rates to copy can be specified by CrvSet in IN format, this is, between brackets, or by Dev, which will be the currency of the fixing rates to copy.

  • Examples:
  • - Copy the fixing rates of fixing curves ESP MIBOR, EUR LIBOR of fixing date 27-JAN-1999 to 29-JAN-1999 with status copied MKTfxgCpFxgRates (Ctx, "('ESP MIBOR', 'EUR LIBOR')", "", "27-JAN-1999", "29-JAN-1999", 0, mkt_sta_copied, 1)
  • - Copy the fixing rates of the ESP fixing curves of fixing date 27-JAN-1999 to 29-JAN-1999 MKTfxgCpFxgRates (Ctx, "", "ESP", "27-JAN-1999", "29-JAN-1999", 0, mkt_sta_copied, 1)
Parameters
CtxDatabase conection
CrvSetFixing curves in IN mode. If no Fixing curves are wanted, empty string.
DevCurrency. If no currency is wanted, empty string.
DatOrgFixing date of the original fixing rates to copy
DatDstDestiny fixing date to copy the fixing rates
IsGvIs Global View flag. Used only to diffusion.
StaStatus to save the values
CpToGvIfNotIf the fixing rates must be copied to GV if there are not yet. This parameter is onlu take into account if IsGv is 0.
Returns
Error code (0: OK)
