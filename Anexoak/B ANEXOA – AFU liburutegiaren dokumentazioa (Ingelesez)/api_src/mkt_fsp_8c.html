Modified
  • Jon BARRUTIA - 14/Jun/2006: Redo the forward price calculation
  • Edurne GORDOA - 23/Jan/2015: Take into account O/N and T/N points to calculate forward price
  • David ROMERO - 20/Apr/2015: Avoid Sonar warning (redundant asignment to itself) for unused parameters and simplify if cases of FwpDev and FwpDem variables.
Algorithm
  • Calculate the swap point of Dev vs. USD:
  • + For O/N type points: (FwdPrx = SpoPrx - (FspTN + FspON) / Pip)
  • + For T/N type points: (FwdPrx = SpoPrx - FspTN / Pip)
  • + The rest (FwdPrx = SpoPrx + Fsp / Pip)
  • Calculate the swap point of Dem vs. USD:
  • + For O/N type points: (FwdPrx = SpoPrx - (FspTN + FspON) / Pip)
  • + For T/N type points: (FwdPrx = SpoPrx - FspTN / Pip)
  • + The rest (FwdPrx = SpoPrx + Fsp / Pip)
  • *Fwp = FwpDem / FwpDev;
  • if (strcmp (Qmr, "DIRECT") == 0) *Fwp = *Fwp;
  • else *Fwp = 1 / *Fwp;
