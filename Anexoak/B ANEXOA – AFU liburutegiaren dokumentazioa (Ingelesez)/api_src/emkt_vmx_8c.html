Treats the modification of the matrix code when inserting volatilities.

Treats the modification of the matrix code when inserting volatilities, that is, in the matrix of volatilities manipulation window.

  • Set as label of the Z axis criterion 'Strike' or 'Duration' depending on the Z axis of the selected matrix
  • If the selected matrix has an equity as subjacent, set the Z axis criterion as invisible
  • If clear the field, the default values are: "Strike" and field visible.
  • Modify the Z axis interactor to set the correct one depending on the concept shown in it.
Parameters
CtxDatabase context
LdObjList of objects of the window.
LdValList with all columns that returns the select sentence (in string format)
InxSelIndex selected in LdVal list
Ner(out). Flag to say that the window will be redrawn
Returns
Error code (0 if OK).
