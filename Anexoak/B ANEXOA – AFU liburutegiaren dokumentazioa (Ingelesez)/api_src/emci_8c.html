Detailed Description

In the MonteCarlo simulation engines there are some specific actions from the buttons and right mouse and change specifics.

int EMCIcnfSav ( void *  LdObj,
char *  LstNam,
int *  Done 
)

Subwindow confirm.

Put the data list of the sub-window into the LdObj list of the father window, that is, the instrument definition window (MciIns). It passes to modify phase the instruments definition window. For the variable parameters window, it marks the instrument as modified and to re-generate the indexes of the instrument. To do it, it sets an object called MCI_RGN_IDX into the objects list.

Parameters
LdObjWindow's objects descriptor
LstNamName of the list to be saved.
Done(OUT) Done
Returns
Error code (0 if OK).
