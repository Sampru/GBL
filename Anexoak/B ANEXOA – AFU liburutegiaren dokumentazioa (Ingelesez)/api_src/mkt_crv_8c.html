Interpolate a value for a date in a curve.

Interpolate a value for a date in a curve

  • Search the first point with default after or equal to Dfl.
  • If there is a point with same default, return its value.
  • If Dfl is between two points, linear interpolation between previous and next point value
  • If Dfl is previous to the first point of the curve with a valid value, depends on the extrapolation method.
  • + Horizontal extrapolation: return the value of the first point.
  • + Extrapolation with one point: create a curve point with 0.0 value for the default of 0.0. Make a straight line between that new point and the first point of the curve and calculate the value where it cuts the requested default.
  • + Extrapolation with two points: make a straight line between first and second point and calculate the value where it cuts the requested default.
  • If Dfl is bigger than last point of the curve with a valid value, depends on the extrapolation method.
  • + Horizontal extrapolation: return the value of the last point.
  • + Extrapolation with one point: return the value fo the last point.

  • + Extrapolation with two points: make a straight line between last point and the point previous to the last one and calculate the value where it cuts the requested default.

    Parameters
    LdCrvList of points.
    EtpMthExtrapolation method. See enum crv_etp_mth for details.
    DflDefault whose value is requested.
    ColValValue column
    ColDflDefault column
    Val(out) Interpolated value
    Returns
    Error code (0 if OK).
    Author
    Basilio TAMAYO - 26/Jul/2007
    Modified
    • Basilio TAMAYO - 27/Jul/2007. Init Dfl2 to correct compilation warning.

Interpolate a value for a date in a curve.

Interpolate a value for a date in a curve

  • Search the first point with date after or equal to Dat.
  • If there is a point with same date, return its value.
  • If the date is between two points, linear interpolation between previous and next point value
  • If the date is previous to the first point of the curve with a valid value, depends on the extrapolation method.
  • + Horizontal extrapolation: return the value of the first point.
  • + Extrapolation with one point: create a curve point with 0.0 value in the begin date of the first point. Make a straight line between that new point and the first point of the curve and calculate the value where it cuts the requested date.
  • + Extrapolation with two points: make a straight line between first and second point and calculate the value where it cuts the requested date.
  • If the date is after the last point of the curve with a valid value, depends on the extrapolation method.
  • + Horizontal extrapolation: return the value of the last point.
  • + Extrapolation with one point: return the value fo the last point.

  • + Extrapolation with two points: make a straight line between last point and the point previous to the last one and calculate the value where it cuts the requested date.
