static int lcEMKToprCalOprVal ( void *  LdObj,
char *  Obn,
int *  Res 
)
static

Verification of option/volatility or vice versa calculation.

Verify if the calculation of the option from the volatility or vice versa must be done. The answer to this verification is given in Res parameter, which can have the following values:

  • 0: not calculate
  • 1: calculate
  • 2: ask for calculation
Parameters
LdObjObject list
ObnData type in object list: Opr, Vol
Res(Out) Verification result. 0: not calculate 1: calculate 2: ask for calculation
Returns
Error Code (0: OK)
