Detailed Description

Montecarlo instrument's schedules definition management.

Acronyme
MCIsch

Montecarlo instrument's schedules definition management

  • Object: Sch, list of montecarlo instruments schedules definition
  • Name Type Origin Description
  • SchCod CTR_TYPE afuMciSch.SchCod Schedule code
  • MciCod CTR_TYPE afuMciSch.MciCod Montecarlo instrument code
  • SchAst CUR_TYPE afuMciSch.SchAst Asset
  • SchPer INT_TYPE afuMciSch.SchPer Periodicity
  • SchUnt char (2) afuMciSch.SchUnt Periodicity unit
  • SchBdt CDA_TYPE afuMciSch.SchBdt Begin date
  • SchEdt CDA_TYPE afuMciSch.SchEdt End date
  • SchDay INT_TYPE afuMciSch.SchDay Day of expectation
  • SchDayOrd INT_TYPE afuMciSch.SchDayOrd Ordinality in the week
  • SchMan char (3) afuMciSch.SchMan Manually inserted or not
  • Author
    Edurne GORDOA - 13/Jan/2004
