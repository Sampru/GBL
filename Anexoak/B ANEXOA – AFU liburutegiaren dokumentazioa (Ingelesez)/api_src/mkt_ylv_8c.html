Calculate the stripped volatilities for a given curve on caplet periods.

This function makes a stripping process on a given spot volatility curve. The resultant list consists on several points that represent the caplets correspondent to the volatility curve, each with its stripped volatility. This stripped or forward volatility cannot be calculated directly from the spot volatility, and thus an iterative estimation is performed until the correct stripped volatility is found (further explained on code)

Parameters
CtxDatabase context
LdYldCrvYield curve (can be different from LdAsoYldCrv)
LdYlvCrvYield volatility curve
LdAsoYldCrvAssociated yield curve for the yield volatility curve
DevCurrency
DatCalculation date
MrgMargin on which the stripped volatility is estimated
PcsPrecision. The bigger the number, the bigger (+ 1 decimal) the precision
LdRes(out) Stripped volatility list with caplet periods
Returns
Error code (0: OK)
