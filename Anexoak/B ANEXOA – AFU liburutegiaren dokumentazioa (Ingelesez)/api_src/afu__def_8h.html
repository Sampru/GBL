Get a set of bucket dates for a treatment and currency.

Get a set of bucket dates for a treatment and currency. If there are no dates for the given currency, it gets the dates for the treatment without currency. The returned list has the following columns:

  • Ech: Number of days for the maturity
  • Mat: Maturity, e.g. 2
  • Unt: Unit of the maturity, e.g. "Y"
  • Stm: Maturity string from the maturity and unit
