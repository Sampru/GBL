Complete a date taking another date as a reference.

Complete a date taking another date as a reference, according to a certain method Note that all dates, intro and output dates, are in user format.

Parameters
DateBufCompleted date
DateInDate to complete (user format)
RefDateReference date (user format)
FmtDate style
  • 5 = European
  • 10 = American
MethodMethod used. This method is used to complete the date
  • 1 (DatFllMthLwrEqu): date <= refDate
  • 2 (DatFllMthGrtEqu): date >= refDate
  • 3 (DatFllMthGrt): date > refDate
Returns
  • Error code (0: Ok; 1: Incorrect date; 2: Correct date, but does not follow Method)

Get the day rank from the day name.

Get the day rank from the day name. Given the day name, this is, "Monday", "Mon", "Tuesday", "Tue", ... it returns the day rank, this is, 1 for Monday/Mon, 2 for Tuesday/Tue, 3 for Wednesday/Wed, ...

  • The day name can be given in long format ("Monday") or short format ("Mon"). See parameters

DayRnk(OUT) 1 for Monday/Mon, 2 for Tuesday/Tue, 3 for Wednesday/Wed, 4 for Thursday/Thu, 5 for Friday/Fri, 6 for Saturday/Sat, 7 for Synday/Sun, 0 if the day name is not a valid one.
