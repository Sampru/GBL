int ALUwbsXtrColFrmLdPar2Ld ( void *  LdPar,
t_AluWbsDstCol *  ArrCol,
void **  LdOut 
)

Extract a list of columns from parameter list to a dynamic list.

The purpose of this function is to unify in a single list all the parameters of a web services that are related to a single list of data and to have the data in the original type. For example, to get a unique list with all the parameters related to a matrix, and to have DYN_DATE column type for date data types.

The programmer will specify destination column type. When the destination is DYN_STRING a specific length can be specified for the resultant column. Otherwise the length of the original column will be taken into account.

To specify the last column of the array, set NULL in the the ClnOri.

Parameters
LdParParameter list
ArrColColumns descriptor
LdOut(out) Resultant dynamic list with the specified columns
Returns
Error code (0: OK, 1: invalid type conversion)
