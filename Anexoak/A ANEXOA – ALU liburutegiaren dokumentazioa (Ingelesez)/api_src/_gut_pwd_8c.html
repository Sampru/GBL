Check a password entered from the entry window.

Check a password entered from the entry window. It is different from GUTpwdVer because at this moment we cannot use any global variable since the connexion has not been stablished yet.

  • If there is any error, we set an error/warning.

  • In case the password if expired, it is the function that calls the responsible to see if an error/warning must be set.

    Algorithm

    See if the password has to be controlled with internal or external password

    Parameters
    CtxDatabase context
    UsrUser to check the password
    GrpGroup the user belongs to
    PwdPassword to see if is correct
    PwdOk(out) Flag, 0 if the password is correct 0, -1 in other cases
    OutDat(out) Flag if the password is out of date (1) or not (0)
    Returns
    Error code (0: OK)
    Author
    Basilio TAMAYO - 09/Sep/2002
