Detailed Description

VIeWing data managing functions.

Header file
viw_def.h
Library
Eng

Viewing data managing functions. These function provide a generic framework to manage viewing windows and their functions.

  • In a view type windows, LdVal list DYN_DATE column type datas are in RDB format, not in user.
  • The code in this file drives the interface on its duty. Decisions must be taken at this level, so that if the interface generator (now ILOG VIEWS) is changed everything need not be reimplemented.
  • Details:
  • A new viewing screen NEW without particularities normally requires:
  • - Add the call to the menu descriptor(s) MENU. File: mnu_dat_MENU.dat
  • - Add its general information. File: viw_des_alldata.dat
  • - Create a descriptor with criterium information File: viw_crt_NEW.dat
  • - Create a descriptor with data information File: viw_dta_NEW.dat
  • And optionally:
  • - Add new enumerated types. File: enums.dat
Author
Edurne GORDOA - 28/May/1998
