Detailed Description

Semaphore management.

This module provides some basic functions to manage Unix semaphores semaphores in a simplified way.

  • The Unix semaphore-handling functions may also be used but if a simplified use is desired it is proposed to add the functionallity to this module.

Create a semaphore and initialise its value.

Create a semaphore and initialise its value

Parameters
KeyThe semaphore key number. All processes using it must know this number.
ValueInitial value of the semaphore. Typically 1 for a Mutex
Returns
Upon success the semaphore identifier is returned, needed by other semaphore functions within this module. Errors: (errno set to indicate the error)
  • -1: Cannot create semaphore.
  • -2: Cannot assign initial value.

int Psemaphore ( int  Semid)

Carry out a P operation over an existing semaphore.

Carry out a P operation over an existing semaphore. A P operation sets the executing process to sleep on a list associated with the semaphore when the semaphore value is 0. Otherwise it decrements in one the semaphore value in one.

Parameters
SemidSemaphore identifier, as returned by CreateSemaphore1 or UseSemaphore
Returns
  • 0: ok
  • -1: error (errno set to indicate the error)

int Vsemaphore ( int  Semid)

Carry out a V operation over an existing semaphore.

Carry out a V operation over an existing semaphore. A V operation awakes the first process sleeping on the list associated with the semaphore. If no process is waiting the semaphore value is incremented in one.

Parameters
SemidSemaphore identifier, as returned by CreateSemaphore1 or UseSemaphore
Returns
  • 0: ok
  • -1: error (errno set to indicate the error)
