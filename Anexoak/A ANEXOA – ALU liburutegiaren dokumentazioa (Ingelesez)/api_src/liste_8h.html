Detailed Description

===========

IMPORTANT :

Acronyme
LS
  • avant l'utilisation des fonctions de ce module, vous devez avoir fait
  • appel a la fonction MMinit() dans votre programme principal (cf le
  • module memoire.h).
  • Ce module definit un objet de type liste ainsi que toutes les fonctions
  • permettant de travailler sur la liste et sur ces elements.
  • Dans ce module, l'element le plus a droite est considere comme le
  • dernier element de la liste, et l'element le plus a gauche comme le
  • premier element de la liste.
  • Remarque :
  • L'element contenu dans la liste peut etre de tout type, de la chaine de caracteres a la structure.
  • ATTENTION :
  • Le bon fonctionnement des services fournis n'est garanti que si les
  • fonctions sont utilisees en respectant le mode d'emploi donne ci-dessous.
Author
J-Pascal CHEVIN - DD/Mar/1993.
