Specific confirm of the window of specific path definition.

Specific confirm of the window of specific path definition

  • Get data in matrix. In matrix there are two lines for each treatment, first one for the path for production/development version and the second one for path for integration version
  • Add path in odd lines into the integration column of previous line and remove them.
  • Remove all current paths in database

  • Save in database the paths in matrix.

    Parameters
    CtxDatabase context
    LdObjObject list
    Done(out) If confirm is finished or generic confirm is required.
    Returns
    Error code (0: OK)
