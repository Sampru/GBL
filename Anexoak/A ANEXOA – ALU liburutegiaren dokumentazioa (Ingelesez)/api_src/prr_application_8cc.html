aluPrr application main program

aluPrr application main program. It accepts four execute modes:

  • DefMod –> Default mode, launch aluPrr application
  • PrnFilMod –> Print file mode. It prints indicated file from indicated or default printer
  • WinFilMod –> Window file mode. It shows indicated file in a window
  • WinMod –> Window mode. It shows file selector dialog and if any file is selected, it opens a window
  • GenPdfFil –> Generate and print a pdf file.
