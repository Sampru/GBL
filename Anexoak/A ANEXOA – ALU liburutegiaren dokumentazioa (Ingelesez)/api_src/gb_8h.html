Les parametres renseignent les specifications de conversion des chaines de format sqlVerb, sqlWhere, sqlOrder et sqlGroup.

Parameters
ctxDatabase context
numFieldsNombre de field selectionnes
sqlVerbRequete SQL
sqlWhereWhere clause
sqlOrderOrder by clause
sqlGroupGroup by clause
...The pairs followed by the SELECT stmt, WHERE stmt, ORDER_BY stmt GROUP_BY stmt, and the parameters to all of these
