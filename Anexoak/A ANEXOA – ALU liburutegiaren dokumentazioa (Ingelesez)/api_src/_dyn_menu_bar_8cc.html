Create a new label from a label and a mnemonic.

PROTOTYPES OF STATIC FUNCTIONS

Create a new label from a label and a mnemonic, this is, the new label will contain the old label but with the a caracter \^ before the letter which represents the mnemonic.

  • Example:
    • Label -> "File"
    • Mnemonic -> 'F'
    • NewLabel -> "\^File"
Parameters
LabelThe label where the mnemonic is going to be set
MnemonicThe mnemonic of the label
NewLabelThe new label which will contain the old label with the mnemonic set
Returns
void.
