ALUMCR_EXP int FVWListInit ( void **  Ld,
char *  BaseDir,
char *  IniDir,
char *  Filter 
)

Create a dynamic list with files and directory list.

Create a dynamic list with columns:

  • IniDir : Initial directory. It's the value of the label TEMPORARY
  • BaseDir : Directory where the subdirectories are
  • DirAct : Current directory where the report files are
  • FileAct : Current report file.
  • Filter : Pattern to select files on DirAct
  • Recent : Flag:
    • ALL_FILES: See all files on DirAct
    • MOST_RECENT: Only see the most recent files on DirAct
  • DirList : List of directories on BaseDir
  • FileList: List of files on DirAct
Parameters
Ld(out) Resulting dynamic list
BaseDirInitial value for BaseDir column. It can be a environment variable
IniDirInitial value for IniDir column. It can be a environment variable
FilterInitial value for Filter column
Returns
Error code (0: OK)
