Detailed Description

Ilog Generic Utilities header file.

Ilog generic utilities header file ILOG generic windows included as a ALU module utility are:

  • IGUrsc : Resources window to change application's colours and fonts values
  • IGUsqt : SQL extractor window to execute SQL statements in an application
