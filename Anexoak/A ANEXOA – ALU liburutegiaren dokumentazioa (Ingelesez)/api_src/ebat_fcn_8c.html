Get the list of all dependences with external list of dependences.

Get the list of all dependences with external list of dependences

  • For a application and batch, we provide the list of dependences of its tasks (LdExt)

  • For the rest of dependences, get them from database

    Parameters
    CtxDatabase context
    BatBatch name
    LdExtExternal list of dependences of tasks of App and Bat
    LdDep(OUT) List of tasks dependences grouped by tasks: DepBat: Batch of dependant task DepNam: Name of dependant task LdMst: List of master tasks
    • MstBat: Batch of the master task
    • MstNam: Name of the master task
    • Mth: the way dependant task depends on master task (see enum BatMthTyp)

    Error code (0: OK)
