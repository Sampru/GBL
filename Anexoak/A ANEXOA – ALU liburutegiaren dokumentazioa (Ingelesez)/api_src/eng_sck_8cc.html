IlvAny  UseArg

static

Handler for the open socket.

Handler for the open socket It treates the messages received in the open socket. Now, there are the following generic messages:

  • message: shows an alert to the user
  • question: asks a question to the user
  • refresh: with this messages there are two possibilities:
    • - VIW: launches the refresh action for the windows defined in configuration file
    • - MIE: launches the specified treatment for the windows defined in configuration file
Parameters
Timnot used. Necessary because it's a callback format
