Constructor of InquireString class.

Constructor of InquireString class. The InquireString class derivates of IlvDialog class, so the the constructor of this last class will be execute before this one.

  • This constructor creates a new dialog which recovers a string entered by the user. This dialog can be of two types:
  • - if (Type == 0), it is the default value, the string which the user will introduce will be normal.
  • - if (Type == 1), the string which the user will introduce will be a password string, so the string will appear codified.
  • - if (Type == 2), the string which the user will introduce will be a password string, so the string will appear codified. Also if need introduce the user
  • Actions:
    • - Register callbacks
    • - Add the objects to the panel
    • - Init the colours
    • - Init the fonts
    • - Init the look and feel of the objects
Parameters
LabelTitleThe label of the text field which hols the string
TypeType of the dialog. 0: normal string (default value), 1: password string, 2: password and user string
