Open an INET socket and bind it to a given port for READ operations.

Open an INET socket and bind it to a given port for READ operations

Parameters
HostNameSocket hostname. If NULL current hostname.
PortNumberSocket port number. If 0 the system will assign a port number. This number may be retrieved through IPCgetSocketPort ()
SockTypeSocket type. Valid types (include ipc_def.h):
  • SOCK_DGRAM: Datagram socket
  • SOCK_STREAM: Stream socket
SockId(out) Socket identifier. This identifier is the "file descriptor"

ALUMCR_EXP int IPCreadSocketStr ( int  SockId,
void **  MessRead,
int *  NbRead 
)

Read a message from an opened and bound socket.

Read a message from an opened and bound socket.

This function limits the size of the message to 1023 bytes.

It guarantees that in an error free scenario, the byte following the last byte of the message is null ('\0'), makes it suitable for using it with character strings.

It allocates the memory for the message, which must be freed through MMfree.

Parameters
SockIdSocket identifier.
MessRead(out) Pointer to the result (must be MMfree-d)
NbRead(out) Number of bytes read
Note
Error handling is carried out in such a way that no trace is shown under error conditions, but an error message is set, which may be tested through the ERRisThereAnyError () function. These messages should be treated or removed.
Returns
Error code (0: OK)
