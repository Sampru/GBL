ALUMCR_EXP int CTRLuserDate ( char *  UserDate)

Control the correctness of a date in USER format.

Control the correctness of a date in USER format, i.e. "dd-mm-yy" (european, fmt=5) or "mm-dd-yy" (american, fmt=10). This function does not carry out completion. It is aimed at controlling date strings just before running a treatment.

Parameters
UserDateDate in user format
Returns
Error code
  • 0: Date Ok
  • -1: NULL data string pointer
  • -2: Incorrect string length
  • -3: Incorrect separator format
  • -4: Date component(s) not digits
  • -5: Invalid month
  • -6: Invalid day
  • -7: Unregistered error
