Gets all defined criteria.

Returns a list with the properties of defined criteria. If group identifier parameter is -1 all criteria defined are loaded. Otherwise only criteria defined for the hierarchy of the group are loaded. If there is not a hierarchy specific for the group all criteria defined are loaded.

Parameters
CtxChannel to database
TrtTreatment identifier
IdgGroup identifier for which get criteria hierarchie, to get all defined criteria pass -1
LdCrt(OUT) Criteria list
Returns
Error code (0: OK)
