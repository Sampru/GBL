Transcodification of parameters.

Transcodification of some parameters, this is, starting from some criteria to obtain a resultant code.

  • The LlTcd is a list list which must contain the column:
  • + Transcodification code
  • + Sublist with:
  • + Result code
  • + n transcodification criteres

  • The specified columns must be in the specified order. The result code, transcodification code and at least one critere column must exist obligatorily in LlTcd list.

    Parameters
    LlTcdTranscodification list list
    Val(out) Obtained transcoded value
    MulHitFlgFlag on how to behave if there are more than one code that match criteria:
    • 0: Set a warning
    • 1: return the first code that matches.
    NoTcdFlgFlag which says if error must be shown when code is not found (0: No, 1: Yes)
    IdeGroup identifier
    NbRefNumber of referencies for the transcodification
    ...NbRef data. Each datum must be a character data pointer (char *).
    Returns
    Error code (0: OK)
    Author
    Basilio TAMAYO - 11/Nov/2004

Load transcode data for a code/identifier from file.

Load transcode data for a code/identifier from file. It reads data from file and save in database for a code/identifier.

  • The file format will be:
  • - Value
  • - 1st criterion
  • - 2nd criterion
  • - ...
  • - 15th criterion
  • The criteria columns are variable but note that the must exist all the criteria columns until a specific one even their values are null.
  • Lines starting with "#" are considered as comment, they are not considered as data
  • The importation directory of the file is got from:
  • - if there is a valid input for the specific path entry "TcdImp".
  • - otherwise is got from importation directory pointed by FXT_IMPORT environment variable and the aluTcd subdirectory.
  • Once the file is correctly loaded it is renamed to do not insert twice
Parameters
CtxDatabase context
TcdCodTranscode datum code
TcdIdeTranscode datum identifier
MthFile loading method.
  • C: Complete (empty table for current code/identifier).
  • P: Partial (insert new data, modify exisiting ones)
SepFile columns separator
FilNamTranscode filename to load
Returns
Error code (0: OK)
