char* OSDgetEnv ( const char *  EnvVar)

Function which retrieves the current value of the enviroment variable if it exists.

An environment variable has a maximum size limit of 32.767 characters, including the null-terminating character. A static variable is needed in order to preserve the value after the function call. If this function is called twice consecutively, it is needed to appropriately store its content before in order to not lose it. If the environment variable does not exist the function returns NULL.

Parameters
EnvVarThe name of the environment variable to get
Returns
DirNam Content of the environment variable
Author
David ROMERO - 16/03/2015
Modified
