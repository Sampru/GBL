Detailed Description

GE, - Gestionnaire des messages d'erreurs.

Header file
ge.h

Prise en compte des messages dans le fichier err_outil.dat avec appel a GEmessageInfo pour les messages internes a GE

  • The errors message are showed in English. If you want to show them in French you must initialise PRJ_LANG environment variable to 'F'.

    Author
    JP CHEVIN - 23/Mar/1993
