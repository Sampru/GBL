Read data from parametrage file.

Read data from parametrage file defined by GBPARAM enviroment variable.

  • Out parameters memory must be reserved.

  • Trys to read encripted database access parameter file. If encripted file does not exist, try to read not encripted file. When neither of them exits return an error.

    Parameters
    AlsAlias of the database
    Usr(OUT) User
    Pwd(OUT) Password
    App(OUT) Application
    Mch(OUT) Machine
    Srv(OUT) Database server
    Lan(OUT) Language
    Dsn(OUT) Data source name
    Chs(OUT) Character set
    Returns
    Error code 0: OK
