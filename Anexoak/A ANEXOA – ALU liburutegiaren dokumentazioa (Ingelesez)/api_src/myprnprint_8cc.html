Header file
myprnprint.h

Printing window

  • To add a new report:
  • - Define all new criterion in prn_cra_alldata.dat. Every criterion added here must be added in function: RunOneReport
  • - Define report general caracteristics in prn_dsc_alldata.dat
  • - Assign criteria to the report in prn_crt_XXX_XX.dat (XXX_XX report ide).
  • This step is not necessary if a report with the same criteria is already defined. In this case, indicate its identifier in prn_dsc_alldata.dat
  • - Assign the controls to do before launching the report in prn_ctl_alldata.dat
  • Every control added here must be added in function: REPprevControls
