Trace a byte stream.

Trace a byte stream, typically containing special characters.

Printable characters are shown as they are.

For non-printable characters their ASCII code is shown in decimals preceded by a backslash, sourounded in brackets.

Examples:

  • <null>: [\0]
  • <bs>: [\8]
  • <cr>: [\10]
