)

Returns the GUC's keys.

Returns the GUC's keys. They are used to encrypt/decrypt the file where are saved the keys that each application uses to encrypt/decrypt the param.std.

  • It can return only 10 keys because the encrypt/decrypt algorithm only can take into account this number of keys.
Parameters
NbKeyNumber of returned keys
...out). NbKey pointers to the keys. They will be pointers to char (char *) without reserved memory
Returns
Error code (0: OK)

Return the keys to decipher the param.std.

Returns the keys to decipher the param.std, for each application. GUC provides a executable to each application, so that it generates a encrypted file (with own keys of GUC) that contains the keys of param.std. This way we have a encrypted file with the keys of param.std that GUC is able to decipher.
