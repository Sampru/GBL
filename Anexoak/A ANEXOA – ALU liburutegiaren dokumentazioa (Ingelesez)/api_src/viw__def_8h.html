Calculates coordenates X and Y for each criteria.

Calculates coordenates X and Y for each criteria. This funcion is used into the VIW engine and into the MIE engine (for the criteria). To know what criteria are visible we use the column "wex" in the VIW and else, for the MIE, we use the column "vsb". For that, we see if "wex" column exists and if so that means that we are in the VIW engine, else we suppose that we are in the MIE. In VIW/REP we use "wex" and the application as next:

  • if criteria exists in GV and if we are in fox or
  • if criteria exists in FOX and if we are in GV,
  • if criteria exists in other application and if we are in Fox or GV,
  • if criteria not exists, The treated criteria is invisible The function lcVIWgetUbicCrts is executed while the maximum size that the criteria occupy is bigger than the size of the window and the numbers of columns of criteria is bigger than zero. This is a solution for the width of criteria but not for the height. There is not solution for thath by the moment.
Parameters
LdCrtList of criteria
FieldHeigthHeigth for the fields
AppApplication
MaxCrtLineMaximun criteria for each line
MaxWinWidthMaximun window width
Returns
Error code (0 if OK)
