static int lcEEMLsrvAthChg ( void *  LdSdtMtx,
int  LinMtx 
)
static

Change engine function for e-mail server definition window.

Check the value of 'Authentication' column:

  • - if the value is 'N', means that the user and password are not need so these cells must be desactivate for the modified matrix line.
  • - if the value is 'Y', means that the user and password are need so these cells must be activate as obligatory for the modified matrix line.
  • The checking will be done in LinMtx line of the matrix. If it is -1, all the matrix lines will be checked.
Parameters
LdSdtMtxMatrix descriptor list
LinMtxMatrix line that needs the checking. If it is -1, all the lines in matrix will be checked
Returns
Error code (0: OK)
Author
Josune BIKUNA - 28/Oct/2005
Modified
