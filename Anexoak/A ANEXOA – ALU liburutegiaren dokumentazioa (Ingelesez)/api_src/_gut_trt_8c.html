Detailed Description

Function to get a specific import/export path for a treatment.

Header file
gut_def.h
Acronyme
GUTtrt

Function to get a specific import/export path for a treatment. These paths are saved in a ALU table called GutTrtPth. In this table there will be line for the treatments that can treat an specific path.

Returns a command code associated to a treatment.

Returns a command code associated to a treatment. This command code must be one of those configured in 'Commands' table.

Parameters
CtxDatabase context. It will be a FIT MO or FIt BO connection.
TrtTreatment
Cmd(OUT) Pointer to a reserved variable that will contain the command code. Reserved size should be [6 + 1]
Vld(OUT) Validity flag. 0: valid command code; -1: invalid command code.
AddInfFlag that indicates if an information message has to be added in case of the command is not found.
Returns
Error code (0: OK)

Returns a path associated to a treatment. It can be used to read/write files from/to this directory. The path will depend on integration or production/develeompent version
