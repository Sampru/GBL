Detailed Description

Database access specific functions for ORACLE DBMS.

Acronyme
GBdmdOra

File that contains specific functions for access with SQL sentences to a ORACLE DataBase Management System. Important: they will be implemented directelly with the functions existing in gb.c/gbodbc.c, i.e. they will not be directly call to the lower libraries (odbc, ...)

File that contains functions for database data manipulation with calls to specific functionalities of DataBase Management System. Important: they will be implemented directelly with the functions existing in gb.c/gbodbc.c, i.e. they will not be directly call to the lower libraries (client-library of sybase, odbc, ...)
