Create and initialisation of the local variables of the motor.

Create and initialisation of the local variables of the motor

  • Initialisation of the local variables of the motor
  • Creation the list of modes if is necesary. It creates always at the same coordenates.
  • Get the objects of the window and insert values in LdSdt
  • NOTE: All objects have to have a specific name: Ldbn: for label Fdbn: for field Udbn: for unit label
Parameters
NameName of the static to create
CoordXCoordenate X to draw the modes list
CoordYCoordenate X to draw the modes list
displayDisplay of the window
contContainer of the window
SdtIlvIndicates if is created from a generic file (staticData.ilv) or a specific file.
LdObjObjects list
Returns
  • Pointer to the created window
