double CVT10to328 ( double  nb)

Convert a decimal mode price into 328 mode.

Convert a decimal mode price into 328 mode

  • The 328 mode price usually has only 3 decimals, the two first are 32 type and the third is 8/32 mode. This function has been modified to get more that 3 decimals, and in this case the decimals which goes after the third position are decimals of the type 8/32. See explanations in modified part (23/Jul/2002).
Parameters
nbPrice to convert
Returns
Error code (0: OK)
Author
Modified
  • Josune BIKUNA - 23/Jul/2002. Get the 328 mode price with more than 3 decimals. The decimals which goes after the third position are the decimal to the 8. This modification has been done because in CLP (Juantxu) has a problem with a Tbond deal. This deal has been negociated in decimal price and he has an efective. As in FOX the Tbond price is in 328 mode, if only 3 decimals are used in 328 mode, it is not possible to reach to the efective which has been negociated.

Convert a 328 mode price into decimal mode

  • - aaa.xyz = aaa + (xy / 32) + (z / 8 / 32)

This function has been modified to treat more that 3 decimals, and in this case the decimals which goes after the third position are decimals of the type 8/32.
