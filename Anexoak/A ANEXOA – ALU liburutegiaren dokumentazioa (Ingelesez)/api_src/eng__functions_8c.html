Filter window configuration data depending on some values.

Filter window configuration data depending on:

  • - Field identifier: if it is NULL it is not taken into account
  • - 1 criterion: if it is NULL it is not taken into account. Otherwise, it deletes the lines that does not match with the value
  • - Usr: if it is NULL it is not taken into account. Otherwise, it deletes the lines that does not macth with the value
  • - Grp: if it is NULL it is not taken into account. Otherwise, it deletes the lines that does not macth with the value

Export data to a file.

Export data of a dynamic list into a file. Depending on a data descriptor and a data list it exports data to a file.

  • The data descriptor list must have at least next columns:
  • + vsb: saying which data are visible
  • + dec: number of decimals in the case of float
  • If the Hed parameter is "Yes", the column labels will be inserted in the first line of the generated file.
  • The LdObj list must have an object called ExpFilTyp with the type of engine that has called to this functionality.

  • Characters "\\ / | \" < > ; ? *" have to being removed from file name.

    Parameters
    CtxContext
    LdObjObject list
    FileNameName of the file that will be created/modified.
    HedData with header (Yes/No)
    NumFmtFormat type for numeric values.
    DatFmtFormat type for date values.
    FilFmtFormat of file (XML, Plain text, ...)
    FilExtFile extension
    Returns
    Error code (0: OK)
