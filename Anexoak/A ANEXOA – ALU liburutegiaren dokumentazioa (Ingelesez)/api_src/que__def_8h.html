ALUMCR_EXP int QUEsetLd ( void **  LdObj,
QueTypes  Type,
char *  QueId,
char *  Question,
void *  LdAns 
)

Inserts a question in the object list.

Inserts a question in the object list. The inserted question will be another object 'Question' in LdObj. The looK and feel of the question will depend on the parameter Type, which can have the values:

  • - QUE_YES_NO
  • - QUE_SINGLE
  • - QUE_MULTI
  • The possible answers must be in LdAns. This list must have obligatory a column 'ans' with the answers, and optionally another column 'mrk' which will be used to mark selected answers. If 'mrk' column does not exist in LdAns will be created.
  • The list LdAns will be destroyed when calling OBJdestroy, so the user must not destroy it.
Parameters
LdObjPointer to an object list
TypeType of question
QueIdQuestion identifier
QuestionLabel of the question
LdAnsList of answers
Returns
Error code (0: OK)
