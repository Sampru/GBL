Static data importation of data coming in a dynamic list.

Static data importation execution getting data from a dynamic list. It follows the same function execution schema than the user interface. That means:

  • - It loads the descriptors associated to the static data concept
  • - For each line to insert:
  • + Initalise static data descriptor key field values
  • + Executes the static data reading function
  • + If there is a record in database with the same keys:
  • - Initialises the descriptor with read record field values
  • - It launch the specific reading function
  • - We have modify mode
  • + otherwise:
  • - We have entry mode
  • + Initialise static data descriptor fields with new data to insert
  • + Executes the RETURN key action in visible static data fields
  • + Launch the controls
  • + Executes the confirmation
  • NOTE: At this moment modifications on the key fields are not supported.
