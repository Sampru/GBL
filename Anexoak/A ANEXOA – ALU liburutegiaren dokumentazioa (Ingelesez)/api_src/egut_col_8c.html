Colors window managements specific confirmation.

Colors window managements specific confirmation

  • If InsAll (insert all) parameter is 1, all colour values are inserted into database, otherwise, if InsAll is 0, the colour values are removed from database and default colours are used.

    Parameters
    CtxChannel to database
    LdObjObject list
    InsAllIndicates if all values must be inserted or no (1: Insert all, 0: don't insert all)
    Dne(OUT) Boolean. All specific confirmation done (1) or generic needed (0)
    Returns
    Error code (0: OK)
