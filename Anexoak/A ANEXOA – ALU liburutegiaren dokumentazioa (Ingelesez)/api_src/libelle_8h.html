int LBloadLibelle ( void  )

Cette fonction initialise et remplit la liste contenant l'ensemble.

Cette fonction initialise et remplit la liste contenant l'ensemble des libelles contenu dans le fichier $PATH_LIB/libelle_frm. La liste n'est pas triee, car le fichier l'est deja.

Returns
0 si tout s'est bien passe -1 si le fichier de libelle n'a pu etre trouve -2 si la variable d'environnement PRJ_LANG n'est pas positionnee ou n'a pas une valeur connue.
