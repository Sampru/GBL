Returns the SQL sentence to convert a string field to datetime (date + hour)

  • These will be the returned values:
  • - for ORACLE, the necessary sentence for converting to 'DD-MON-YYYY HH24:MI:SS'
  • - for the rest, the necessary sentence for converting to 'dd mon yyyy hh:mm:ss:mmm'

int GBsqlFltNbRow ( int  NbRow,
char **  SqlStrSel,
char **  SqlStrWhe 
)

Returns the SQL sentence to recover only N rows from a sentence.

Returns the SQL sentence to recover only N rows from a sentence.

  • The function returns two strings because depending on the SGBD, the sentence that must be use to filter the rows is use in the SELECT or in the WHERE statement. One one them will be returned with the correct string and the other one with an empty string "".
Parameters
NbRowNumber of rows that is wanted to be filtered
SqlStrSel(OUT) Returns the string for the SELECT statement. It will be empty "" in case it is not used for the current SGBD. It must be free.
SqlStrWhe(OUT) Returns the string for the WHERE statement. It will be empty "" in case it is not used for the current SGBD. It must be free.
Returns
Error code (0: OK)
