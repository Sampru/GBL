Phylosophy

The information in a XML file forms a tree structure. The DYN structure generated is a list with the root of the tree and three sublists: – list with the element's child – list with the attributes of the element – list with the element's father

  • The value of an element is stocked in a column of DYN_STRING type
  • The list of LdElm parameter must be one column: – Val
  • To get this list can use the function GUTxmlGetElm. This function get the list of element to read from the table gutCodXml.
  • With this XML file:
  • <TramiteCNMV xsi:noNamespaceSchemaLocation="EsquemaFicheroIICMobiliarios.xsd"
  • xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  • <DocumentoCNMV>
  • <CodigoTramite>EFI</CodigoTramite>
  • <IndicadorFirma>DF</IndicadorFirma>
  • </DocumentoCNMV>
  • <EstadosFinancierosIIC>
  • <DatosIdentificativosEntidad>
  • <Gestora>
  • <Tipo>SGIIC</Tipo>
  • <NumeroRegistro>2.3</NumeroRegistro>
  • <Denominacion>aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa</Denominacion>
  • <EstadosGestora>
  • <Estado_MA7>
  • <Unidades clave="0010">0</Unidades>
  • </Estado_MA7>
  • </EstadosGestora>
  • </Gestora>
  • </DatosIdentificativosEntidad>
  • </EstadosFinancierosIIC>
  • </TramiteCNMV>
  • - The DYN structure generated is:
  • - Column Atr: pointer to list with the attributes
  • - Column Chd: pointer to list with the childs of the element
  • - Column Fth: pointer to list with the father of the element
  • - Column Elm: name of the element
  • - Column Val: value of the element (is a string)
  • - This is the result of the function GUTxmlFmlRes
