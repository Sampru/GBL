int ENGccSetStatus ( void *  LdObj,
char *  Msg 
)

Set a message in the statusbar of an engine window.

Set a message in the statusbar of an engine (SDT, MIE) window. Send a "SetStatus" message only if the Real time between processes cst flag is activated. This function is used in JAVA applications. It is neccesary to know which window must be updated, for it, correspondent window identifier is added in the message, it is treated in MainFrame.java file.

Parameters
LdObjobject list
Msgmessage to display in the statusbar
Returns
Error code (0: OK)

Get the message inserted in the ERR module list and resets the list.

Get the message that has been inserted in the ERR module list. If there is no error a null is returned. This function returns an object table setting in the 0 position the error type (info, warning, error) and the error message in the 1 position. It resets the error list
