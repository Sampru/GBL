ALUMCR_EXP int MTXstdDes ( MATRIX *  Mtx,
MATRIX *  MtxRes 
)

Calcul the standard deviation matrix.

Calcul the standard deviation matrix. The result matrix dimension for (M x N) matrix is (1 x N) matrix. Where the column values are the standard deviation of the rows.

  • S = standard deviation
  • X = individual score
  • M = mean of all scores
  • n = sample size (number of scores)
  • S = sqrt (SUM (pow (X - M, 2)) / n - 1)

MtxResResult matrix
