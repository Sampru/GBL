Add a line.

Adds a data line. It must add a line in each list that exists in LdSdt, that is, LdVal, LdSta, LdRel, LdObl, "vld", "cgv", "act", "col" and "edt". LdGid is actualised when the matrix is redrawed (creating). Note that a valid data is writed into 'val' column, the engine writes an empty field, that it's a VALID datum.

Parameters
