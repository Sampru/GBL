Book window specific configuration confirmation.

Interface with ALU engines, for book specific configuration confirmation.

  • - Take matrix data
  • - Control of data. If there is a message the function goes out and the confirmation is not executed.
  • - Take selected criteria values
  • - Delete previous lines for current criteria values
  • - When window is not in cancel mode save new data in database

CtxDatabase context
LdObjObject data list
Don(out) 1: specific confirm action done
