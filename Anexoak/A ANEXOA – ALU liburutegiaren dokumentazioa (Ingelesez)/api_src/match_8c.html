Check if a string matches with a pattern.

Check if a string matches with a pattern (Regular Expression).

Parameters
patternPattern to match
  • A pattern may contain:
    • * => matches 0 or more characters
    • ? => matches any single character. NOTE: "??-" is converted into "~" by the compiler!!
    • [set] => matches one character in the set, possible escaped, or with ranges, e.g. a-z
    • [^set] => matches one character NOT in the set (reverse matching)
    • => A leading ] matches with itself. A non-paired [ pattern is a bad pattern
stringString to check
Returns
  • 0: No match
  • 1: Success
  • 2: Incorrect mask
