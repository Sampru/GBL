JNIEXPORT jint JNICALL Java_com_almis_alu_gui_ThreadRTP_THDini ( JNIEnv *  Env,
jobject  Obj 
)

Initialise thread specific functions.

Initialise thread specific functions.

  • In integration and production environments thread traces are send to a log file named with main application's log file with the .Thd.log suffix. By this way both filenames have the same name except the suffix and they can be easily differentiated for each launching. Currently only GB and TM traces are sent separately to each file. It also sets the isolation level if it is configured to do it; depending on a flag: "aluIsoLev"
Global variables
  • AppTyp: In integration and production modes send thread traces to LOG_FIL.Thd.log file
  • LOG_FIL: Log file to which main program traces are redirected.

ObjClass for which convert the list
