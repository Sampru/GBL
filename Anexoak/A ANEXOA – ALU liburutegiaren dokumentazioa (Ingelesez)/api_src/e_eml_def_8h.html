Chech if server identifier is in use or not.

Chech if server identifier is in use or not. If it is in use, set a warning and set 'Dne' flag to one (to not delete the line), else maintain the 'Dne' flag to zero and engine will remove the line.

Parameters

LdObjObject list
InxLinIndex of the matrix where line is added
