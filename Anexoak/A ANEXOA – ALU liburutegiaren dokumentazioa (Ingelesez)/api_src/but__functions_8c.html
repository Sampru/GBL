Get the data from user and execute the wgnuplot application.

Get the data from user and execute the wgnuplot application. To execute the plot application it's neccesary to create two files:

  • "file".gnu : Instructions for plot application.

The name for these files is the name of the window where the plot application is launched. Get all criteria from graphic tool window and depending on these, create distinct types of "file".gnu, for differents graphics or options, and create "file".dat, with the selected axis data.
