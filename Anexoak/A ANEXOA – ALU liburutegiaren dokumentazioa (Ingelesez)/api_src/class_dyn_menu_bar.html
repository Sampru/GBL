void DynMenuBarCallback ( IlvGraphic *  g,
IlvAny  Any 
)

Callback of the menu items.

Callback of the menu items when the menu access rights are being configured. Actions:

  • Get the correspondent menu item code from the list
  • Verify if the menu item code is already stored in access righgts list
  • If it is not stored, add a new line in the access rights list with the information that the menu item is restringed.
  • If it is already stored, modify the line where it is in the way that if it was forbidden, now it is not, and id it was not forbidden, now it is.
Parameters
gIlvPopMenu object of the selected menu item
AnyIndex of the list which stores menu item codes
Returns
void.

This is the constructor of DynMenuBar class, but as this class derivates from IlvMenuBar class, the constructor of IlvMenuBar class is called before this function.

Create a sub menu and set it to a menu item.

Create a sub menu with the data contained in the dynamic list. LdSubMenu and set it to the menu item MenuItem. Before setting the new sub menu to MenuItem, check if any of the items of the new sub menu has another sub menu, and in this case call to this function again to set it.

Parameters
DisplayThe display where the menubar is located
ContainerThe container where the menu bar will be added
MenuItemThe menu item where the new sub menu is going to be set
LdSubMenuA dynamic list which contains the data of the new sub menu
ConfAccRightsDeterminates if the menu is going to be used to configure access rights or not
