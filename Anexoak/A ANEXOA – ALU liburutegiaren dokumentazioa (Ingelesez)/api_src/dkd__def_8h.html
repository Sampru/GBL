Decode an integer number from a substring.

Decode an integer number from a substring.

It decodes an integer from the substring delimited by a beginning position and its size in a string.

If any of the characters of the substring is not a digit it returns an error code.

This function will concentrate on the specified portion of string, so that the rest of the string may contain whatever character.

The maximum integer that fits a long (4 bytes) is 2,147,483,647. It is therefore not authorised a size bigger than 10. A 10 character number bigger than the maximum may give unexpected results (normally negative numbers).
