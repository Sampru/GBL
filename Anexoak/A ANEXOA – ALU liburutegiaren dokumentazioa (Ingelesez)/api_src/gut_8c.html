Get version strings.

Get version strings

  • Database version is read from the CST GenRel
  • Executable version is read from a file called inf<App>VerBf.dat
  • ExeTim is read from the last modification time of the executable defined by an environment variable named EXEC_FILE
Parameters
AppName of the application
DBver(out) Database version string. Must be freed
ExeVer(out) Executable version string. Must be freed
ExeTim(out) Time of last modificacion of the executable
Returns
Error code (0: OK)
