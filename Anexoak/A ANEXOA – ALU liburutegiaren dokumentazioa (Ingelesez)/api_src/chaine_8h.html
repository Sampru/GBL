ALUMCR_EXP char* STRcpwhbl ( char *  dest,
char *  ori,
int  flag,
char *  refe 
)

Effectuer une copie d'une chaine.

Effectuer une copie d'une chaine dans une autre en eliminant les caracteres blancs situes a differentes positions, selon la valeur de flag

  • Cette fonction recopie ori dans dest (on peut avoir ori == dest).
  • Lors de cette recopie, certains caracteres ne sont pas recopies, en fonction de leur appartenance a la chaine refe (si un caractere appartient a refe, il n'est pas recopie), et de leur position dans la chaine ori. Le critere de position est gere grace au flag.
    • valeur de flag effet
  • -3 ne recopie pas les x (debut, fin et milieu de chaine)
  • -2 ne recopie pas les x du milieu et du debut de chaine
  • -1 ne recopie pas les x du milieu et de la fin de chaine
  • 0 recopie comme strcpy
  • 1 ne recopie pas les x de fin de chaine
  • 2 ne recopie pas les x du debut de chaine

  • 3 ne recopie pas les x du debut et de la fin de chaine ou x est un caractere appartenant a refe.

    • Nota : en cas de contradiction (par exemple supprimer les blancs en fin et conserver ceux en debut pour une chaine composee de blancs, c'est la condition sur le debut de chaine qui predomine on considere qu'un caractere blanc est un caractere
    Parameters
    dest
    ori
    flag
    refe
    Returns
    Author
    Modified

Recopie les n premiers octets de Org dans Dst, ou n est la taille de Fmt.

Recopie les n premiers octets de Org dans Dst, ou n est la taille de Fmt. Si Fmt est plus court que Org, la chaine est tronquee. Il est possible que Org == Dst. Si la chaine Dst est NULL, la valeur renvoyee est celle d'un buffer statique
