Parameters

FmtDate style

  • 10 = American

  • 1 (DatFllMthLwrEqu): date <= refDate
  • 2 (DatFllMthGrtEqu): date >= refDate
  • 3 (DatFllMthGrt): date > refDate
Returns
  • Error code (0: Ok; 1: Incorrect date; 2: Correct date, but does not follow Method)

Get the day rank from the day name.

Get the day rank from the day name. Given the day name, this is, "Monday", "Mon", "Tuesday", "Tue", ... it returns the day rank, this is, 1 for Monday/Mon, 2 for Tuesday/Tue, 3 for Wednesday/Wed, ...
