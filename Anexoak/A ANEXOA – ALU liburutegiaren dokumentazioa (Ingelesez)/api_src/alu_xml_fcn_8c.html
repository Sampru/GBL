Transform a TXT file into XML format (2 dimensions)

Only valid to create xml files with 2 dimensions (the dimensions strings must be given by parameters: LblOne, LblTwo).

The resultant xml file will be created in the same folder and filename (with xml extension).

It uses format data lists to be able to specify the format of the datum to be read and exported.

Xml detination format list: If NULL it uses the default format. Useful for columns with more than 30 characters. This format list can be created calling the DYNinitDynFormatList function.

Parameters
LstDefTxt: List format definition table (to read txt source file)
NbFmtTxt: Number of columns in list (txt)
FilNamSrc: Full txt file name (including path and extension)
LstDefXml: List format definition table (to create xml file)
NbFmtXml: Number of columns in list (xml)
NbHdrLin: Number of header lines in txt file (not to be exported to xml)
LblOne: Label in level 1 on XML file (once per data list)
LblTwo: Label in level 2 on XML file (once per data list line)
Returns
Error code (DYN_SUCCES : OK; DYN_ERREUR : error)
Author
Iban VARAS - 28/Mar/2014

int DYNsavXml ( void *  LdDta,
void *  LdFmt,
char *  FilNam 
)

Save a list in a file in XML format.

Save a list in a file in XML format. It generates two files:

  • - FilNam.xml: File containing the data in XML format
  • - FilNam.xsd: File containing the XML schema of the dynamic list
  • It uses a format data list to be able to specify the format of the datum to be exported. If NULL it uses the default format.
  • This formats list can be created calling the DYNinitDynFormatList function.
  • - Call to the function taking into account hierarchy between libraries:
  • - ERRchkDyn (&LdDta, DYNsavXml (LdDta, LdFmt, FilNam));
Parameters
LdDtaData list
LdFmtFormat data list; otherwise, NULL
FilNamFull file name (including path and extension)
Returns
Error code (DYN_SUCCES : OK; DYN_ERREUR : error)
