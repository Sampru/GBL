Save a list in a file in XML format.

Save a list in a file in XML format. It generates two files:

  • - FilNam.xml: File containing the data in XML format
  • - FilNam.xsd: File containing the XML schema of the dynamic list
  • It uses a format data list to be able to specify the format of the datum to be exported. If NULL it uses the default format.
  • This formats list can be created calling the DYNinitDynFormatList function.
Parameters
LdDtaData list
LdCrtCriteria List
LdSchMtxMatrix Schema list (lab, siz, jfy, ide, dbn, vsb)
LdSchCrtCriteria Schema list (lab, siz, jfy, ide, dbn, vsb)
LdHdgHeader list
LdColColour list
FilNamFull file name (including path and extension)
StyNamStyle file name
CrtNamCriteria file name
RepTypType of report (matrix or static)
Returns
Error code (DYN_SUCCES : OK; DYN_ERREUR : error)

FILE *  Fil,
void *  LdFmt,
void *  LdCol,
int *  Typ,
int *  Dec,
char *  Nam 
)

Insert the data from the list to the xml file.

Get the information from the corresponding list (criteria or line) and insert all in the xml file. If a criterion has more than one line, use ";" to separate values. Characters which are special for xml are checked and its own encoding is set. The functions is limited to 512 columns as maximum

Parameters
DYNlistThe dynamic list that has to be inserted in the xml file.
FilPointer to the xml file
LdFmtFormat of the datum into the file
LdColInformation about the colour
TypDYN data type of the datum
DecDecimal numbers of the datum
NamName of the node (criteria or line)
Returns
Error code (0: OK)
