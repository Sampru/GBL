int FMLsetFmtDefaults ( void *  LdDat,
void *  LdFmt 
)

Initialise a format list given the data list.

Initialise a format list, that is, a list specifying the format for each of the columns in the dynamic list. A sample data list is needed to get the default values that depend on the type and information. -+-------------------------------------------------------------------—

  • lab (STRING).
  • Meaning: Column label.
  • Default: Column name in dyn list. -+-------------------------------------------------------------------—
  • dec (INT).
  • Meaning: Number of decimals, only significant for DYN_FLOATs.
  • Default: 6
  • Other initialisations:
  • int FMLinitMinSizes (void *LdDat, void *LdFmt, int MaxDec, int Format) -+-------------------------------------------------------------------—
  • siz (INT).
  • Meaning: Size in characters of the resulting string. Note that the
  • ending null character will be added internally.
  • Default: Depends on type.
  • - DYN_DATE: 8
  • - DYN_STRING: Size in the list
  • - DYN_INT: 12
  • - DYN_FLOAT: 13
  • - DYN_VOID : sizeof (void *) * 2 + 2 (the value is showed in hexadecimal with 0x)
  • Other initialisations:
  • int FMLinitMinSizes (void *LdDat, void *LdFmt, int MaxDec, int Format) -+-------------------------------------------------------------------—
  • mth (INT).
  • Meaning: Rounding method (DYN_FLOAT type only)
  • Valid values:
  • 0: Round nearest
  • 1: Truncate
  • 2: Up next -+-------------------------------------------------------------------—
  • fmt (INT).
  • Meaning: Format
  • Default: Depends on the type:
  • - DYN_DATE:
  • Valid values:
  • - 0, 2: American "mm-dd-yy"
  • - 1, 3: European "dd-mm-yy"
  • Default value: DTgetFmt () == 5? European : American
  • - DYN_FLOAT:
  • Valid values:
  • - 0: American, formatted, e.g. 3,159,124.4238
  • - 1: European, formatted, e.g. 3.159.124,4238
  • - 2: American, no separator for thousands, e.g. 3159124.4238
  • - 3: European, no separator for thousands, e.g. 3159124,4238
  • Default value: 2
  • - DYN_INT: Default value: 2
  • 0: American, formatted, e.g. 3,159,124
  • 1: European, formatted, e.g. 3.159.124
  • 2, 3: No separator for thousands, e.g. 3159124 -+-------------------------------------------------------------------—
  • jfy (INT).
  • Meaning: Justify. Default value: depends on the type.
  • Valid values:
  • - 0: None
  • - 1: Left
  • - 2: Middle
  • - 3: Right
  • Default value: Depends on type.
  • - DYN_DATE: None
  • - DYN_STRING: Left
  • - DYN_INT: Right
  • - DYN_FLOAT: Right -+-------------------------------------------------------------------—
  • fix (INT).
  • Meaning: If column scrolls
  • Valid values:
  • - 0: Fixed
  • - 1: Scrolling
  • Default value: 1st column Fixed, rest Scrolling. -+-------------------------------------------------------------------—
  • rmr (INT).
  • Meaning: Remove replicated.
  • Valid values:
  • - 0: False (do not remove)
  • - 1: True (remove)

    - Default value: True for 1st column, rest False.

Parameters
LdDatdynamic list with data to take the information for default values
LdFmtformat list, see contents and meaning in general description.
Returns
Error code (0: OK)
