ALUMCR_EXP int ITFgetGraphicResources ( int  grs,
void **  LdGrs 
)

Gets the graphic resources set for the specified set.

Gets the graphic resources set for the specified set. Actually the graphic objects that the engines manages are:

  • winTit: Title of the window
  • mtxTit: Title of the matrix
  • mtxDta: Cells of the matrix
  • butLab: Buttons labels
  • butTrtLab: Treatment button label
  • fldLab: Window fields label
  • fldDta: Window fields datas
  • untLab: Unit label
  • mnuBar: Menu bar
  • staBar: Status bar
  • scrolling: Scrollingin popup objects By the moment the engine manage only the fonts.
  • The fonts associated to each graphic resource set are explicited in deda.doc document. The font and colors of all other objects are fixed. ATTENTION !! Actualise the number of graphic resources ("NbGrs" variable) each time that we add a new resource.
Parameters
grsGraphic resources set
LdGrs(OUT). List of resources
Returns
Error code (0: OK)
