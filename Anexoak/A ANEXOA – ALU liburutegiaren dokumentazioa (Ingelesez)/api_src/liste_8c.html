Detailed Description

Une liste est composee d'une entete et d'un certain nombre d'elements. Chaque element est relie a ses 2 voisins par 2 pointeurs. L'entete contient des informations generales sur la liste (nombre d'elements etc) et un pointeur vers les 2 elements extremes. Les fonctions de ce module considerent qu'une liste egale a nulList est equivalent a une liste contenant 0 element.
