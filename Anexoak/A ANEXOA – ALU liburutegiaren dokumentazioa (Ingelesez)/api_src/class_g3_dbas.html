Component getLaunchableComponent ( )
inline

This method implements Launchable, and should only be called if this demo has not been launched as a stand alone application. The Component returned is a reference to this - a JPanel - if the non-Java3D implementation is being used. Otherwise, a JFrame is constructed with this as its content pane (this is done since the Java3D implemenation is a heavyweight).

Note that the init(boolean) should be called prior to this method.
