Detailed Description

TM, Time handling functions.

Header file
tm_def.h
Acronyme
TM, Time handling functions

This module contains several utilities for time and performance measuring. It has not been created for its inclusion in an application but this could be possible without much modifications.

  • For Windows platform GLBvmmPcsXxx functions have been used to manage correctly global variables for each process. So, for each process has its own global variables.
  • For the correctness use of current module lcTMlogTrc function must be used for the printable traces
    • To print the system time: TMlocalTime ()
    • This function is actually provided as an implementation demo, as more specific functions are often required.
    • The basic utilisation is measuring the time elapsed between two points:

    (1) Initialise

    TMinitClock ()

    (2) Report time elapsed since initialisation (lcTMlogTrc)

    TMprintElapsedTime ()

    • One of its most practical aspects is the manipulation of time lists:

    (1) Initialisation

    TMinitTimeList ();

    (2) Add time marks

    TMaddTimeMark ("Start"); ... (code) TMaddTimeMark ("First"); ...

    (3) Report times elapsed (lcTMlogTrc)

    TMreportTimes (); (4) Destroy

    TMdestroyTimeList ();

  • - To use this module in threads and send TM module's traces to different log files the following functions must be used. (1) Stablish the log file to which send each thread traces TMsetLogFil (FilNam);

    (2) End TM module by each thread TMend ();
