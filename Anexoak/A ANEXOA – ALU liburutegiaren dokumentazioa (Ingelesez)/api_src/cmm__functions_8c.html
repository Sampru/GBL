Detailed Description

User connection and communications.

Header file
cmm_def.h

This module handles communication between users executing different instances of the application.

  • - Table: par_con
  • - Description: Connection parameters
  • - par_con
  • - usr_r char (16) not NULL, -> Receiver (User)
  • - por int not NULL, -> Port
  • - sta char (1) not NULL, -> Status
  • - srv SRV_TYPE NULL, -> Server
  • - pid int NULL, -> Process identifier
  • - usr_s char (16) NULL, -> (not used) Sender (User)
  • - num int NULL, -> (not used)
  • - PRIMARY KEY (usr_r, por)
  • - UNIQUE NONCLUSTERED INDEX par_conI1 (usr_r, pid)
  • - UNIQUE NONCLUSTERED INDEX par_conI2 (por)
  • Author
    Isma ETXEBERRIA - 17/Aug/1999
