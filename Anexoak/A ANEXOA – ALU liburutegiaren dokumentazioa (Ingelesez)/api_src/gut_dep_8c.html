Check closed loops in a list starting with a key.

Check closed loops in a list starting with a key

  • Given a list that have key and secondary columns in a way that one key have one or several secondary values. At the same time, those secondary values are also keys that lead to other secondary values. There is a chain of values that may lead to the initial key value.

  • Note that it will only detect circular loops that start with a specific key

    Parameters
    LdDynamic list
    Clp(out) Flag if there is a closed loop
    NbColNumber of columns of the key
    ...NbCol triplets (InxKey, InxSec, Key)
    • InxKey: Column index of key values in which the data is to be searched for
    • InxSec: Column index of secondary values
    • Key: Pointer to the element searched for, of the appropriate type
    Returns
    Error code (0: OK)

Get closed loops in a list starting with a key.

Get closed loops in a list starting with a key

  • Given a list that have key and secondary column in a way that the key have one or several secondary values. At the same time, those secondary values are also keys that lead to other secondary values. There is a chain of values that may lead to the initial key value (loop).
  • Note: Only smallest chains are kept in the resultan list of loops.
Parameters
LdDynamic list. It can have more columns than key and secondary.
InxKeyColumn index of key values in which the data is to be searched for
InxSecColumn index of secondary values
LdLop(out) Resultan list.
Returns
Error code (0: OK)

Delete duplicated loops in a list.

Delete duplicated loops in a list, when we say loop, we are talking about all data in a line of the list. It looks for data in all columns of the line and then if are equal to the datas in other lines. Only one of repeated loop are kept in retourned list.
