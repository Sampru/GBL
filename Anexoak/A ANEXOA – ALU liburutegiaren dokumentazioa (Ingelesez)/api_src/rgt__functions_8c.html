Detailed Description

Object registration module.

Acronyme
RGT, object ReGisTration

Object registration module

  • This module contains a static list of the registered objects of the applications. This list has the following structure:
  • Type: Object type. Normally, it will be an engine's type: mie, viw, ...
  • Identifier: Identifier of the object
  • Object: Pointer to the object
  • Status: Status of the object. See the possible values in rgt_def.h (tRtmSta type)
  • This module provides functions to initialise/destroy the registered objects list, and also, to change the status of them
