Set the correct entry type for table description and synopsis.

Set the correct entry type for table description and synopsis. If a valid entry type is given as parameter, set it to table description and synopsis, otherwise, set the entry ttype depending on the window mode: in 'entry' mode -> 'introduction' type, otherwise -> 'specific' lookup type
