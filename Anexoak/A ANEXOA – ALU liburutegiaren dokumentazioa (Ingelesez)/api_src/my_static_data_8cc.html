Creates a static data window.

Creates a static data window. Pending SetObjSrc parameter value, it sets or not the source window object list into the current object list. We can access to source object getting "LdObjSrc" object from current window object list.

Parameters
IdeIdentifier of the static data window
LdPossible list to help to create a specific objects list
SdtIlvIndicates if it read a file create by ILOG or not (0: No, 1:Yes)
ModalIndicates if a modal/not modal window will be created.
SetObjSrcIndicates if source window object list must be keep in current window object list
Returns
  • Error code (0 if OK)
