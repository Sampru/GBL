Detailed Description

Access for specific functionalities to a database management system depending.

Header file
gb.h
Acronyme
GBdmd
Library
gb

File that contains functions for database data manipulation with calls to specific functionalities of DataBase Management System. Important: they will be implemented directelly with the functions existing in gb.c/gbodbc.c, i.e. they will not be directly call to the lower libraries (client-library of sybase, odbc, ...)

Bulk insertion in the database. Transactions are allowed.

This function inserts in bulk the list Ld into the table. It can be used within a a transaction. Notes: – The fields with an empty string ("") will be inserted into the table as NULL – The columns to insert are indicated with the parameter NbCol and the next ones. – The option "select into/bulkcopy" must be activated into the database. – The version "High Speed" of the function is used, without LOG of the data, to insert data into the temporary table. – Data into the table are inserted with the SQL sentence INSERT, so the triggers are lunched. – The checkpoints can be indicates with the parameter NbBat – If an error happens, the list is printed in the log

Parameters
CtxContext of database. This is the context where the transaction exist
LdDynamic list
TabTable
NbBatNumber of lines in a batch of send
NbColNumber of columns to insert (0 means all the columns)
...List of indexes of columns to insert (used if NbCol in not 0)
Returns
Error code (GB_SUCCES or GB_ERREUR)

Algorithm
  • - Connect to the same database with other connexion
  • - Create a temporary table with the same struct that 'Tab'
  • - Insert the list into the temporary table with the bulk copy
  • - Return to the original connection
  • - Insert into the table the information from the temporary list
  • - Return to the new connection
  • - Delete temporary table
  • - Delete the new connecion and return to the original one
