Detailed Description

Script manager.

Scripts management taking into account RDBMS (Sybase, Oracle, SQL Server) NOTE: Take into account that if this executable is regenerated, it must be manually moved to Almis command directory.

  • Requiered parameters:
  • + Ekf : Encrypter key file to decipher parametrization file
  • + Rdbms : Relational DataBase Management System. It can be either Sybase (syb), Oracle (ora) or SQL Server (sqs).
  • + Alias : Database alias over is going to do the specified action.
  • + Script: Script file name with full path. Script for aplication in database.
  • + Param : (Optional) Database connection parametration file. In this files all databases should be configured. This file can be encripted.

  • If no file or location specified, it'll take local directory's parametrization file.
