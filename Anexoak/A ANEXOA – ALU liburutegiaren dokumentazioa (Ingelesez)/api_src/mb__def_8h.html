ALUMCR_EXP int MBOOKgetTypAls ( void *  Ctx,
void *  LdAls,
int  Typ,
void **  LdRes 
)

Limit a list of alias to those books of a given type.

Limit a list of alias to those books of a given type taking into account each database type (defined by cst.ide = 'IsGv').

  • Connect to all alias, get the database type and compare it with the specified type. It returns the aliases if:
    • Typ == MBdffAll
    • Typ == MBdffTdb && IsGv == 0
    • Typ == MBdffNdf && IsGv == 1
  • Otherwise, the book is removed from the list.

  • Be careful when using this function because if acces to the 'cst' table of all books. If this function is called into a transaction and this table is locked, there is a dead lock.

    Parameters
    CtxDatabase context
    LdAlsOriginal list of alias. If NULL, it recovers the standard list for diffusion
    TypSpecified book type. It can be: MBdffAll,MBdffTdb,MBdffNdf
    LdRes(out) Resulting list of alias. It can be NULL
    Returns
    Error code (0: OK)
    Author
