Detailed Description

Database access functions using ODBC.

Header file
gb.h acronym : ODBC

Database access functions using ODBC. It uses the ODBC version 3.52.

  • Take into account that in the connections to Oracle databases in the ORA_STR_SET_NLS_LNG macrodefinition the NLS_LANGUAGE parameter is set to ENGLISH because in the Calendar utility dates are treated in english format among other things.
Author
Itziar BIKENDI - 02/Apr/2000

Cette clause peut contenir des references absolues ou relatives. Une reference relative est une reference a la liste dynamique. La fonction accepte un nombre variable d'arguments. Il s'agit tout d'abord des numeros de colonnes servant de reference, et ensuite des numeros des colonnes qui seront mises a jour.

La table test2 est mise a jour a partir d'une liste issue d'une selection sur test1. Pour chacune des lignes, une requete de mise a jour est envoyee. Les champs a mettre a jour sont numerotes 0 et 2, soit reel et string. Pour chaque ligne, la clause where est generee a partir de la valeur des champs dans la ligne. Dans l'exemple ci-dessus, si la colonne entier a pour valeur 10, la clause where sera: where entier = 10 and ville like 'P' La colonne numero 1 (entier) est une colonne de reference. NOTA : si au niveau de la clause where, on souhaite utiliser un '' (par exemple like 'X'), il convient de le doubler, comme dans toute chaine de format.
