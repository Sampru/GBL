Insert the data from the list to the xml file.

Get the information from the corresponding list (criteria or line) and insert all in the xml file. If a criterion has more than one line, use ";" to separate values. Characters which are special for xml are checked and its own encoding is set. The functions is limited to 512 columns as maximum

Parameters
DYNlistThe dynamic list that has to be inserted in the xml file.
FilPointer to the xml file
LdFmtFormat of the datum into the file
LdColInformation about the colour
TypDYN data type of the datum
DecDecimal numbers of the datum
NamName of the node (criteria or line)
Returns
Error code (0: OK)
