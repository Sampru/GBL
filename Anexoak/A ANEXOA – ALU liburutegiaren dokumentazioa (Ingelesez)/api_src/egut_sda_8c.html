Initialise a View descriptor for comparing static data window.

View descriptors initialisation from static data. It takes into account engine's book specific configuration for the window for which load descriptors. The auditory window has the following columns:

  • Modification data: date, time, user and action
  • Modification values: Modified field label, old value, new value.
  • Fields that are Key on window static data
